# 6STEP 업데이트 가이드

## 업데이트 원칙

1. **URL 보존** — 기존 포스트를 수정한다. 새 글로 재발행하지 않는다 (기존 URL의 SEO 가치 유지)
2. **핵심 키워드 유지** — 기존 글의 핵심 키워드와 검색 랭킹을 유지한다
3. **추가 > 삭제** — 기존 콘텐츠를 보강·재구성하는 것을 우선한다. 삭제는 명백한 오류/부정확한 정보에 한한다
4. **최신화 표시** — 글 상단에 업데이트 뱃지를 삽입하여 독자에게 최신 정보임을 알린다

---

## STEP 1: 도입부 리엔지니어링

### 현재 문제

기존 글 대부분이 아래 패턴이다:

```
"안녕하세요, 염용주 변호사입니다. 오늘은 ~에 대해 알아보겠습니다."
```

→ 스마트블록 미리보기에 핵심 정보가 노출되지 않음
→ AI 브리핑이 답변을 추출할 수 없음
→ 인사말로 3줄을 소진한 뒤에야 본론이 시작됨

### 업데이트 방법: 3단 도입부 구조

기존의 긴 인사말을 아래 **3단 구조**로 교체한다.
기존 내용을 삭제하는 것이 아니라, 같은 분량을 더 효율적으로 재구성하는 것이다.

---

**[3단 도입부 구조]**

```
┌─ ① 마이크로 인사 + Direct Answer (1~3줄)
│    인사 1문장 + 정의형 답변 + 핵심 수치/법령 → 한 호흡으로
│
├─ ② 공감 + 손실회피 (1~2줄)
│    독자 감정 인정 → 방치 시 불이익 암시
│
└─ ③ 글 안내 + 목차
     → "아래에서 ~를 다룹니다" + 글의 전체 목차(TOC)
     → 목차 직후 바로 본론 진입
```

---

### ① 마이크로 인사 + Direct Answer

기존처럼 인사말과 본론을 분리하지 않고, 한 문단으로 통합한다.
이렇게 하면 스마트블록 미리보기 3줄 안에 인사와 핵심 답변이 모두 담긴다.

**템플릿:**

```
안녕하세요, 교육법 전문 변호사 염용주입니다.
{글의 핵심 질문에 대한 정의형 답변 1~2문장}.
{관련 법령 조항 + 핵심 수치/기한}.
```

**작성 규칙:**
- "안녕하세요" 인사는 1문장(15자 이내)으로 압축
- 인사 직후, 같은 문단에서 바로 정의형 답변으로 이어짐
- "~란, ~입니다/~됩니다" 종결형 사용 (AI 브리핑 추출 최적)
- 법령 조항 번호, 구체적 수치(기한, 인원, 금액)를 반드시 포함
- 첫 3줄(120자 이내)에 핵심 답변이 완결적으로 제시되어야 함

### ② 공감 + 손실회피

핵심 답변 직후, 1~2문장으로 독자의 감정을 인정하고 "지금 대응하지 않으면 어떤 불이익이 있는지"를 짧게 암시한다.

**템플릿:**

```
{독자 감정 인정 문장}. {방치/지연 시 불이익 1가지 암시}.
```

**작성 규칙:**
- 독자가 이 글을 검색한 심리 상태를 반영 (불안, 당황, 분노, 막막함 등)
- 감정을 먼저 인정한 뒤("갑작스러운 통보에 놀라셨을 겁니다")
- 행동하지 않을 때의 손실을 한 줄로 암시("초기 대응을 놓치면 징계가 확정될 수 있습니다")
- 과장이나 공포 조성이 아닌, 실무 기반의 사실적 경고여야 함
- 변호사 광고 규정 준수: 성과 보장 표현 금지, 사실에 기반한 주의 환기만 허용

**손실회피 패턴 예시:**

| 카테고리 | 감정 인정 | 손실 암시 |
|---------|---------|---------|
| 교원 징계 | "갑작스러운 징계 통보에 당혹스러우실 겁니다" | "소명 기회 전 준비가 결과를 좌우합니다" |
| 학교폭력 | "아이 일이라 더 마음이 급하실 겁니다" | "학폭위 개최 전 72시간이 가장 중요한 대응 시간입니다" |
| 학교안전사고 | "아이가 다쳤다는 소식에 정신이 없으셨을 겁니다" | "초기 증거 확보를 놓치면 보상 청구가 어려워질 수 있습니다" |
| 행정소송 | "부당하다는 생각에 답답하실 겁니다" | "제소 기한(90일)을 넘기면 다툴 수 없게 됩니다" |

### ③ 글 안내 + 목차 → 본론 진입

공감/손실회피 직후, 글의 구조를 안내하고 **목차(TOC)를 배치**한다.
목차는 글의 최상단(썸네일 바로 아래, ①② 직후)에 위치하므로
독자가 원하는 섹션으로 바로 이동할 수 있고, 체류시간이 증가한다.

**템플릿:**

```
아래에서 ①{세부 주제 1} ②{세부 주제 2} ③{세부 주제 3}를
실무 경험을 바탕으로 정리해 드리겠습니다.

**목차**
1. {세부 주제 1}
2. {세부 주제 2}
3. {세부 주제 3}
```

**작성 규칙:**
- 목차는 ②(공감+손실회피) 바로 다음, 본론 시작 직전에 배치
- 목차 항목은 클릭/스크롤 이동 가능한 앵커로 구성 (네이버 에디터 기능 활용)
- 3~5개 항목이 적정 (너무 길면 오히려 이탈 유발)
- 목차 직후 바로 본론 첫 번째 섹션이 시작됨 (추가 인사말 없음)

### 예시 (사립학교 교원 징계위원회)

**Before (기존 도입부):**
```
안녕하세요, 교육법 전문 변호사 염용주입니다.
오늘은 사립학교 교원 징계위원회에 대해 알아보겠습니다.
(중략)
```

**After (3단 도입부 적용):**
```
📌 2026년 2월 기준 업데이트

안녕하세요, 교육법 전문 변호사 염용주입니다.
사립학교 교원 징계는 사립학교법 제61조에 따라 학교법인이 설치한 징계위원회에서 심의합니다.
징계 처분에 불복할 경우 처분이 있음을 안 날부터 30일 이내에 교원소청심사위원회에 소청심사를 청구할 수 있습니다.

갑작스러운 징계 통보에 당혹스러우실 겁니다. 소명 기회 전 준비가 결과를 좌우합니다.

아래에서 ①징계위원회 구성과 절차 ②소청심사 청구 방법 ③실무 대응 전략을
정리해 드리겠습니다.

**목차**
1. 사립학교 징계위원회, 어떤 경우에 열리나요?
2. 징계 처분에 불복하려면? 소청심사 절차
3. 징계 대응, 핵심 전략은?
```

## STEP 2: 본문 구조 재설계 (GEO + 체류시간)

### 2-1. 소제목 체계 점검

기존 소제목이 단순 나열형인 경우, **검색 의도(intent)별**로 재편한다.

**변환 규칙:**

| 기존 패턴 | 개선 패턴 | 스마트블록 매칭 이유 |
|----------|----------|-------------------|
| "징계위원회란?" | "교원 징계위원회, 어떤 경우에 열리나요?" | 질문형 → 스마트블록 '관련 질문' 매칭 |
| "징계 종류" | "교원 징계 종류와 각 처분의 실질적 효과" | 구체화 → 검색 의도 정확히 매칭 |
| "대응 방법" | "징계 통보를 받았을 때 첫 72시간 대응 전략" | 시간 프레이밍 → 긴급성 반영 |
| "관련 법령" | "2026년 기준 교원 징계 관련 법령 정리" | 최신화 시그널 → 신뢰도 강화 |
| "마무리" / "정리" | "실무에서 자주 묻는 질문 (Q&A)" | Q&A형 → AEO 매칭 |

**소제목 재편 원칙:**
- 질문형 또는 검색 의도 반영형으로 전환
- 핵심 키워드를 소제목 앞쪽에 배치
- 각 소제목이 독립된 스마트블록으로 노출될 수 있는 수준의 자족성 확보

### 2-2. GEO 팩트 블록 삽입

각 주요 섹션에 **근거법령·대상·기한·위반시·확인일**을 정리한 팩트 블록을 삽입한다.

**팩트 블록 템플릿:**

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
📋 팩트 체크
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
▸ 근거법령: {법률명 제X조}
▸ 적용대상: {대상}
▸ 기한: {기한 또는 기간}
▸ 위반 시: {제재 또는 불이익}
▸ 확인일: 2026년 {월}월 기준
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

**삽입 기준:**
- 법령이나 제도를 설명하는 섹션마다 1개
- 기존 본문에 이미 팩트 블록이 있으면 내용을 최신화
- 최소 2개, 최대 4개 삽입

### 2-3. Q&A 블록 삽입

관련 질문 2~3개를 본문 중간~하단에 배치하여 AEO 최적화를 강화한다.

**Q&A 블록 템플릿:**

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
❓ 자주 묻는 질문
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Q. {질문 1}
A. {직접 답변 2~3문장. 법령 조항·수치 포함}

Q. {질문 2}
A. {직접 답변 2~3문장}

Q. {질문 3}
A. {직접 답변 2~3문장}
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

**질문 설계 원칙:**
1. 독자가 **실제로 검색할 질문**을 사용한다 (전문 용어보다 구어체)
2. 답변은 **직접적 답변 → 근거 → 실무 팁** 순서로 작성한다
3. AI가 추출할 수 있도록 **Q → A 구조를 명확히** 유지한다
4. 기존 본문에서 다루지 않은 **보완적 질문**을 우선 선택한다

## STEP 3: 리텐션 강화 장치

### 3-1. 훅(Hook) 강화

글의 첫 화면(스크롤 없이 보이는 영역)에 독자를 사로잡는 훅을 배치한다. 기존 글에 훅이 약하거나 없는 경우 보강한다.

**훅 유형 (1개 이상 적용):**

| 유형 | 패턴 | 예시 |
|------|------|------|
| 위기감 | "~를 놓치면 ~이 제한됩니다" | "소청 기한(30일)을 놓치면 불복 수단이 크게 제한됩니다" |
| 수치 | "실제로 ~%가 ~합니다" | "실제로 교원 징계 사건의 약 40%가 초기 대응 부족으로 불리해집니다" |
| 공감 | "~한 상황, 충분히 이해합니다" | "갑작스러운 징계 통보에 당혹스러운 심정, 충분히 이해합니다" |
| 실익 | "이 글을 읽으면 ~를 알 수 있습니다" | "이 글에서 징계 대응의 핵심 3가지를 정리해 드리겠습니다" |

**주의:** 위기감 훅은 **사실 기반**이어야 하며, 공포 조장(광고규정 위반)과 구분한다.
- ✅ "소청 기한은 30일입니다" (사실)
- ❌ "당장 대응하지 않으면 큰일납니다" (공포 조장)

### 3-2. 시각 요소

본문 내 **약 800자마다** 시각적 요소를 삽입하여 체류시간을 높인다.

| 요소 | 용도 | 삽입 위치 |
|------|------|----------|
| 팩트 블록 | 법령·기한 정리 | 법령 설명 섹션 |
| 비교표 | 2개 이상 옵션 비교 | 유형·방법 설명 섹션 |
| 실무 포인트 박스 | E-E-A-T 강화 | 각 본론 중간 |
| 주의사항 박스 | 독자 주의 환기 | 절차·기한 관련 섹션 |

**실무 포인트 박스 템플릿:**

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
💼 실무 포인트
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
{실무 경험에서 나온 조언 1~2문장}
{관련 근거 또는 구체적 수치/기한}
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

### 3-3. 내부 링크

관련 블로그 글 **3~5개**를 본문에 자연스럽게 삽입한다.

**⚠️ 필수: 내부 링크 데이터 소스**
- **반드시 `.claude/skills/naver-blog-writer/references/blog-posts.json` 파일을 참조**하여 실제 발행된 글의 제목과 URL만 사용할 것
- JSON 파일에 존재하지 않는 글은 **절대 링크하지 않는다**
- 카테고리가 동일하거나 관련성이 높은 글을 우선 선택

**배치 규칙:**
- **1/3 지점** (본론1 하단): 관련 기본 개념 글 1~2개
- **2/3 지점** (본론2 하단): 관련 절차/사례 글 1~2개
- **하단** (결론 위): 관련 심화 글 또는 시리즈 글 1개

**링크 형식:**
```
▸ 관련 글: [{글 제목}]({URL})
```

기존 글에 내부 링크가 이미 있으면 유지하되, 부족하면 보충한다.

### 3-4. 목차 추가

Direct Answer 블록 아래에 클릭 가능한 목차를 추가한다.

**목차 템플릿:**


## STEP 4: 법령·판례 최신화

### 4-1. 최신화 체크리스트

기존 글에서 인용된 모든 법률·판례·수치를 점검한다.

| 점검 항목 | 확인 내용 | 조치 |
|----------|----------|------|
| 법률 조항 | 현행법과 조문 번호·내용 일치 여부 | 불일치 시 현행법 기준으로 수정 |
| 판례 | 판례 번호·날짜·결론 정확성 | 오류 시 정정 또는 최신 판례로 대체 |
| 수치/통계 | 인용된 통계의 출처·연도 확인 | 2년 이상 경과 시 최신 데이터로 교체 |
| 제도/절차 | 현행 제도와 설명 일치 여부 | 변경된 제도는 현행 기준으로 수정 |
| 광고규정 | 변호사 광고에 관한 규정 (2025.6.30 개정) 준수 | 위반 표현 발견 시 즉시 수정 |

### 4-2. 최신화 뱃지

글 최상단(Direct Answer 블록 위)에 업데이트 뱃지를 삽입한다.

```
📌 2026년 {월}월 기준 업데이트
```

### 4-3. 법령 변경 이력 박스

해당 글 주제와 관련된 법령이 변경된 경우, 변경 이력을 정리한다.

**법령 변경 이력 템플릿:**

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
⚖️ 법령 변경 이력
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
▸ 기존: {구 법률명 제X조} — {구 내용 요약}
▸ 현행: {현 법률명 제X조} — {현 내용 요약} (시행일: YYYY.MM.DD)
▸ 영향: {변경이 독자에게 미치는 실질적 영향 1~2문장}
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

법령이 변경되지 않았으면 이 박스는 삽입하지 않는다.

## STEP 5: 전환(CTA) 요소 강화

### 5-1. 3단 CTA 배치

| 위치 | 강도 | CTA 유형 | 예시 |
|------|------|---------|------|
| 본문 1/3 | 소프트 | 정보 제공형 | "이 부분에 대해 더 궁금하시다면, 아래 관련 글도 참고해 보세요" |
| 본문 2/3 | 미디엄 | 경험 연결형 | "비슷한 상황에서 어떻게 대응했는지, 실무 경험을 바탕으로 안내해 드리겠습니다" |
| 하단 | 하드 | 전환 유도형 | 사건 유형별 맞춤 CTA (아래 참조) |

### 5-2. CTA 작성 원칙 (광고규정 준수)

**허용:**
- "전문가의 조력이 필요하시다면 언제든 연락 주세요"
- "사안을 정리해 주시면 절차와 방향을 함께 검토해 드리겠습니다"
- "경험을 바탕으로 함께 해결 방안을 찾아드리겠습니다"
- "상담을 통해 구체적인 대응 방안을 안내해 드리겠습니다"

**금지 (변호사 광고에 관한 규정 2025.6.30 개정):**

| 금지 유형 | 금지 표현 예시 | 위반 사유 |
|----------|-------------|----------|
| 결과 보장 | "반드시 승소", "기각 시 100% 환불" | 결과 보장 금지 |
| 수임료 할인 | "수임료 할인", "착수금 특가" | 수임료 할인 홍보 금지 |
| 무료 상담 강조 | "무료 상담", "무료 법률 상담" | 과잉 광고 |
| 공포 조장 | "당장 대응하지 않으면 큰일납니다" | 공포 유발 |
| 최상급 표현 | "최고의 변호사", "유일한 전문가" | 비교 광고 |
| 타 변호사 비하 | "다른 변호사와 다르게...", "타 사무소보다" | 비방 |
| 전문 표현 남용 | "OO 전문 변호사" (주로 취급하지 않는 분야) | 제7조 위반 |

### 5-3. 실무 포인트 박스 (E-E-A-T Experience 강화)

각 본론에 1개 이상의 실무 포인트 박스를 배치하여 경험 기반 전문성을 강화한다.

**실무 포인트 박스 템플릿:**

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
💼 실무 포인트
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
제가 실무에서 자주 마주하는 사례인데요,
{구체적 실무 경험 기반 조언 1~2문장}.
{관련 근거법령 또는 실무적 팁}
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

**작성 원칙:**
- 실제 실무 경험을 바탕으로 작성 (모호한 일반론 지양)
- 구체적 수치·기한·절차를 포함
- 광고규정 위반 표현 절대 불포함
- 자기 자랑이 아닌 독자에게 유용한 정보 제공 목적
