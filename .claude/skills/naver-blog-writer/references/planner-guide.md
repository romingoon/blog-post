# 기획팀 가이드 (Planner Guide)

## 글 구조 및 분량 기준

### 키워드 경쟁도별 분량 기준

| 키워드 경쟁도 | 권장 글자수 (공백 제외) | 이미지 수 |
|-------------|---------------------|----------|
| 낮음 (월 검색량 500 미만) | 2,000~2,500자 | 5~8개 |
| 중간 (월 검색량 500~3,000) | 2,500~3,000자 | 8~12개 |
| 높음 (월 검색량 3,000 이상) | 3,000~4,000자 | 10~15개 |

### 섹션별 분량 기준 (중간 경쟁도 기본)

| 섹션 | 분량 (공백 제외) | 핵심 요소 |
|------|-----------------|-----------|
| 서론 | 250~300자 | 인사말 + 공감 + 시의성 + 주제 소개 |
| 본론1 | 500~600자 | 기본 개념/법적 근거 설명 |
| 본론2 | 500~600자 | 구체적 절차/사례/Q&A |
| 본론3 | 500~600자 | 실질적 대응 전략/주의사항 |
| 결론 | 200~250자 | 요약 + 전문성 언급 + CTA |
| **총합** | **2,500~3,000자** | |

※ 높음 경쟁도일 경우 각 섹션 분량도 비례하여 확대한다 (예: 본론 각 650~800자)

### 문단 구성 원칙 (모바일 최적화)
- 한 문단 **최대 3~4줄** (모바일 기준)
- 중요 정보는 **굵은 글씨**로 강조하되 남용 금지
- **3스크롤(모바일 기준 약 800자) 이내에 핵심 정보** 배치
- 긴 문장보다 짧고 명확한 문장 선호
- 중요한 정보는 문단 첫 문장에 배치

## H2/H3 아웃라인 설계

### 모드별 H2 아웃라인 패턴

| 모드 | H2 구조 | 예시 |
|------|---------|------|
| **전문 정보** | ①개념/법적근거 → ②절차/사례 → ③대응전략 → 마치며 | "교원 징계란?" → "어떤 절차로?" → "핵심 전략은?" |
| **초입문자 가이드** | ①하지 말 것 → ②해야 할 것 → ③왜 그런지 → ④다음 단계 | "절대 하지 마세요" → "지금 바로 해야 할 것" → "왜 이렇게 해야 하는지" → "다음 단계는?" |
| **성공 사례** | ①상황설명 → ②법적쟁점 → ③대응전략 → ④결과 → ⑤시사점 | "어떤 상황이었나요?" → "핵심 쟁점은?" → "어떻게 대응했나요?" → "결과는?" → "배울 점" |

### 권장 H2 제목 구조 【전문 정보 모드】

| 섹션 | H2 제목 패턴 | 예시 |
|------|-------------|------|
| 본론1 | "[주제]란 무엇인가요?" / "[주제]의 법적 근거" | "교원 징계란 무엇인가요?" |
| 본론2 | "[주제], 어떤 절차로 진행되나요?" | "교원 징계, 어떤 절차로 진행되나요?" |
| 본론3 | "[주제] 대응, 핵심 전략은?" | "교원 징계 대응, 핵심 전략은?" |
| 결론 | "마치며" | "마치며" |

### H3 소제목 원칙
- 질문형으로 작성 (AEO 최적화)
- 핵심 키워드 포함
- 독자의 궁금증을 직접 반영
- **스마트블록 분류 키워드와 일치**하도록 작성

## 제목 패턴

### 모드별 제목 패턴

| 모드 | 제목 스타일 | 예시 |
|------|-----------|------|
| **전문 정보** | 정보형/질문형/숫자형 (아래 5가지 패턴 참조) | "교원 징계, 어떻게 대응해야 할까요?" |
| **초입문자 가이드** | 상황묘사형, 1인칭/질문형, 감정 검색어 포함 | "아이가 학폭 통보를 받았는데 뭘 해야 하나요?" |
| **성공 사례** | 사례 기반 + 교훈형 | "교원 징계 취소 사례로 보는 소청심사 핵심 포인트" |

### 전문 정보 모드 제목 패턴 5가지

제목은 **25자 이내**, **핵심 키워드 포함**, 클릭 유도 표현을 사용한다. **5개 제목 추천**을 제공한다.

| 패턴 | 형식 | 예시 |
|------|------|------|
| 정보형 | [키워드] + 핵심 정보 + 감정 어필 | "학폭 가해자 조치, 꼭 알아야 할 것들" |
| 질문형 | "[키워드], 어떻게 해야 할까요?" | "교원 징계, 어떻게 대응해야 할까요?" |
| 숫자형 | "[키워드] 대응 N가지 방법" | "소청심사 승소 5가지 핵심 전략" |
| 문제해결형 | "[키워드] 고민 해결하는 법" | "학생부 정정 거부, 해결하는 법" |
| 최신성형 | "20XX년 [키워드] 달라진 점" | "2026년 학폭법 달라진 점 총정리" |

## 이미지 배치 계획

블로그 글에는 총 5개의 이미지를 배치한다.

| 위치 | 이미지 성격 | 목적 | 권장 스타일 |
|------|-----------|------|------------|
| 서론 직후 | 주제를 상징하는 대표 이미지 | 첫인상, 주제 전달 | 넓은 구도, 밝은 조명 |
| 본론1 중간 | 개념 설명 보조 이미지 | 이해도 향상 | 미디엄샷, 전문적 분위기 |
| 본론2 중간 | 사례/상황 묘사 이미지 | 공감 유도 | 따뜻한 톤, 인물 중심 |
| 본론3 중간 | 해결책/대응 관련 이미지 | 행동 유도 | 집중된 구도, 결단력 있는 분위기 |
| 결론 전 | 희망적/긍정적 분위기 이미지 | 안심, 신뢰 형성 | 밝은 자연광, 희망적 분위기 |

### 추가 이미지 유형 가이드

기본 5개 외에 키워드 경쟁도에 따라 추가 이미지를 배치한다:

| 추가 이미지 유형 | 위치 | 예시 |
|---------------|------|------|
| 절차 플로우차트 | 본론2 | 학폭위 진행 절차 다이어그램 |
| 핵심 정보 요약 카드 | 각 본론 말미 | "꼭 기억하세요" 요약 이미지 |
| 관련 법령 하이라이트 | 본론1 | 법 조항 핵심 부분 강조 이미지 |
| 체크리스트 이미지 | 결론 | 대응 체크리스트 인포그래픽 |

### 이미지 규격/스타일
- **썸네일**: 1300 x 885px (네이버 썸네일 최적 사이즈)
- **본문 이미지**: 가로 폭 최소 960px 이상
- **색감**: 신뢰감을 주는 네이비/블루 계열 + 따뜻한 포인트 컬러
- **텍스트 오버레이**: 본문 이미지에 핵심 키워드 텍스트 오버레이 권장

## 키워드 리서치 절차

글 기획 전 다음 절차로 키워드를 조사한다:

1. 네이버 키워드 도구(또는 블랙키위 등)로 **검색량 확인**
2. **메인 키워드 + 세부 키워드(롱테일) 2~3개** 선정
3. 해당 키워드의 **상위 5개 포스팅 분석**:
   - 글자수, 이미지 수, 구조 파악
   - 빠져 있는 정보(정보 갭) 식별
4. **연관 검색어 / 자동완성 키워드** 수집

### 초입문자 모드 검색어 매핑

초입문자가 실제로 검색하는 키워드는 **롱테일(long-tail)이며 감정적**이다. 기존 전문 콘텐츠와 초입문자 콘텐츠가 서로 다른 검색어를 잡도록 설계한다:

| 기존 전문 콘텐츠가 잡는 키워드 | 초입문자 콘텐츠가 잡아야 할 키워드 |
|------------------------------|--------------------------------|
| 학교폭력 사안처리 절차 | 아이 학폭 가해자 어떡해 |
| 교원 징계 절차 가이드 | 교사 고소당하면 어떻게 |
| 학교안전사고 법적 책임 | 학교에서 아이 다쳤을 때 |
| 교원소청심사 기초 | 징계 불복 방법 |
| 사교육카르텔 대응 전략 | 사교육카르텔 연루 교사 어떡해 |

**초입문자 키워드 선정 원칙:**
- 1인칭/구어체 질문 형태 ("~어떡해", "~어떻게", "~뭘 해야")
- 감정 키워드 포함 ("막막", "당황", "걱정", "처음")
- "처음", "첫 대응", "뭘 해야" 등 초보자 시그널 키워드 우선

## 리서치 요청사항 작성 가이드

기획팀은 리서치팀에 전달할 **조사 요청 리스트**를 작성한다:

1. **핵심 키워드** — 메인 키워드 1~2개, 서브 키워드 3~5개
2. **필수 조사 항목** — 관련 법률, 최신 판례, 통계 데이터, 최근 뉴스
3. **조사 우선순위** — 시의성 높은 항목부터
4. **타깃 독자** — 학부모/교사/학생 등 구체적 대상

## 타깃 독자 분류 기준

기획 시 다음 4가지 타깃 독자 유형 중 주요 대상을 선정한다. 주제에 따라 1~2개 유형이 주요 타깃이 된다:

| 타깃 유형 | 특성 | 주요 관심사 | 주제 예시 |
|----------|------|-----------|----------|
| **학부모** | 자녀의 교육 문제에 관심이 높음 | 학교폭력 대응, 학생부 기재, 입시 관련 법률 | 학폭 피해 대응, 학생부 정정 |
| **교사/교원** | 징계, 소청, 권리 보호에 관심 | 교원 징계 대응, 성비위 무혐의, 직위해제 | 교원 징계, 소청심사 |
| **학생** | 본인의 권리와 절차에 관심 | 학교폭력 가해/피해 절차, 전학 조치 | 학폭 재심, 전학 불복 |
| **학교법인/사립학교** | 기관 차원의 법적 리스크 관리 | 교원 채용/해임, 감사 대응, 사학 분쟁 | 사학 분쟁, 교원 해임 |

## 의뢰인 의사결정 여정 (Decision Journey)

블로그 글이 잠재의뢰인의 의사결정 과정에서 어떤 단계를 겨냥하는지 기획 단계에서 명시한다:

| 단계 | 의뢰인 상태 | 블로그 콘텐츠 역할 | 콘텐츠 앵글 예시 |
|------|-----------|------------------|----------------|
| **인식** (Awareness) | "나에게 법적 문제가 있구나" | 문제 정의 + 공감 | "교원 징계란 무엇인가요?" |
| **탐색** (Consideration) | "어떤 방법이 있지?" | 선택지 비교 + 절차 안내 | "소청심사 vs 행정소송, 무엇이 다를까?" |
| **결정** (Decision) | "이 변호사에게 맡겨도 될까?" | 전문성 증명 + 사례 | "교원 징계 사건, 실무에서 본 핵심 포인트" |

**기획 시 적용:**
- 콘텐츠 플랜 출력 형식에 `### 타깃 의사결정 단계` 필드를 포함한다
- 대부분의 블로그 글은 **인식~탐색 단계**를 겨냥 (정보 제공 콘텐츠)
- **결정 단계** 겨냥 글은 사례 중심 + 전문성 증명 비중을 높임

## 시리즈/연재 전략

3,000자로 부족한 주제는 시리즈로 기획한다:

- 하나의 큰 주제를 **3~5편**으로 분리
  - 예: "학교폭력 완벽 가이드" → (1)개념편 (2)절차편 (3)대응편 (4)판례편
- 각 편은 **독립적으로 완결**되면서도 상호 내부 링크로 연결
- 각 글 서론에 시리즈 안내, 결론에 다음 편 예고

---

## 발행 전략

- **권장 발행 시간**: 오전 6~8시 또는 오후 12~14시 (직장인 통근/점심 시간)
- **발행 빈도**: 주 2~3회 (꾸준한 활동성이 C-RANK에 긍정적)
- **주제 집중도**: 교육법/학교폭력 관련 글이 전체의 **70% 이상** 유지
