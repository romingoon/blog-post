# SEO팀 가이드 (SEO/AEO Optimizer Guide)

## 네이버 C-RANK 알고리즘 (신뢰도)

C-RANK는 블로그의 **전문성과 신뢰도**를 평가하는 네이버 고유 알고리즘이다.

### 최적화 기준
- 글의 주제에 대한 **전문가로서의 깊이와 신뢰도**가 드러나도록 작성
- 핵심 키워드를 제목, 서론, 본론, 결론에 걸쳐 **자연스럽게 배치**
- 제목과 첫 문단에 핵심 키워드 포함
- H2, H3 태그를 활용한 구조화
- **법령, 판례, 정부 자료** 등 공신력 있는 출처 인용

### 키워드 밀도 규칙
- ⚠️ **특정 키워드 20회 이상 반복 금지** (스팸 판정 위험)
- 자연스러운 동의어/유의어 활용으로 키워드 밀도 조절
- 키워드 스터핑(stuffing) 절대 금지

## 네이버 D.I.A+ 알고리즘 (체류시간)

D.I.A+는 **사용자의 체류시간과 반응**을 측정하는 알고리즘이다.

| 전략 | 적용 방법 | 예시 |
|------|----------|------|
| 훅(Hook) | 첫 문장에서 독자 관심 사로잡기 | "갑자기 학폭위 통지서를 받으셨나요?" |
| 의외성 | 통계나 인사이트 제공 | "실제로 80%의 학부모가 이 사실을 모릅니다" |
| 스토리텔링 | 사례 기반 내러티브 | "최근 상담 사례를 통해 설명드리면..." |
| 질문 유도 | 중간중간 질문 던지기 | "그렇다면 어떻게 대응해야 할까요?" |
| 동기부여 | 독자가 끝까지 읽도록 유도 | "결론에서 핵심 체크리스트를 정리해 드립니다" |

### D.I.A+ 최적화 체크포인트
- [ ] 서론 첫 문장에 훅(Hook) 있는가?
- [ ] 본문 중 의외성 있는 통계/인사이트가 포함되었는가?
- [ ] 사례 기반 스토리텔링이 활용되었는가?
- [ ] 독자에게 질문을 던지는 부분이 있는가?
- [ ] 끝까지 읽도록 유도하는 동기부여 문장이 있는가?

## SEO (검색 엔진 최적화)

### 핵심 SEO 규칙
- **제목**: 25자 이내, 핵심 키워드 앞 배치
- **서론 첫 문단**: 핵심 키워드 필수 포함
- **소제목**: 질문형으로 작성 (예: "~는 어떻게 해야 할까요?")
- **메타 설명**: 100~150자, 핵심 가치 제안 명확히
- **내부 링크**: 관련 이전 포스팅 2~3개를 본론2/본론3에서 자연스럽게 링크

### 키워드 배치 전략
| 위치 | 키워드 배치 규칙 |
|------|----------------|
| 제목 | 핵심 키워드를 가장 앞에 배치 |
| 서론 첫 문단 | 핵심 키워드 1~2회 자연스럽게 포함 |
| H2 소제목 | 각 소제목에 핵심/서브 키워드 포함 |
| 본문 | 문단당 키워드 1~2회 자연스럽게 분산 |
| 결론 | 핵심 키워드 1회 이상 포함 |
| 메타태그 | 핵심+서브+관련 키워드 모두 포함 |

## 네이버 스마트블록 최적화

네이버는 VIEW 탭을 폐지하고 **스마트블록(Smart Block)** 시스템으로 전환했다. 검색 결과가 주제 의도별로 분해되어 여러 블록으로 나뉘어 표시된다.

### 핵심 원칙
- 하나의 글에서 **하나의 명확한 검색 의도**에 답변하도록 작성
- 소제목이 **스마트블록 분류 키워드**와 일치하도록 설계
  - 예: "학폭 가해자 전학 조치"를 다룰 때 → "전학 조치 요건", "전학 조치 불복 절차", "전학 조치 사례" 등 세부 의도별 소제목 구성
- **첫 문단 2~3줄 이내에 핵심 답변 제시** (스마트블록 미리보기에 노출)

### 스마트블록 체크포인트
- [ ] 글 전체가 하나의 명확한 검색 의도에 집중하는가?
- [ ] 소제목이 스마트블록 분류 키워드와 일치하는가?
- [ ] 첫 문단 2~3줄에 핵심 답변이 포함되어 있는가?

### 초입문자 모드 스마트블록 전략

초입문자 콘텐츠는 "**기본 정보**" 또는 "**첫 대응**" 블록에 노출되도록 설계한다.

**제목 최적화:**
- 제목에 "처음", "첫 대응", "어떻게"를 포함
- 감정 검색어("막막", "당황", "걱정")를 제목 또는 서론에 배치

**미리보기 최적화:**
- 첫 2~3문장에 **직접적 답변** 배치 (스마트블록 미리보기에 노출)
- 예: "학폭 가해자 통보를 처음 받으셨다면, 48시간 내에 해야 할 5가지가 있습니다."

**소제목 최적화:**
- H2 소제목을 **독자의 실제 질문** 형태로 작성
- 예: "지금 뭘 해야 하나요?", "비용은 얼마인가요?", "변호사가 꼭 필요한가요?"

## AEO (Answer Engine Optimization)

AI가 정보를 쉽게 이해하고 답변으로 활용할 수 있도록 최적화한다.

### AEO 핵심 전략
- Q&A 형식을 활용하거나 질문 형태의 소제목 활용
- **"무엇", "어떻게", "왜"** 질문에 직접 답변하는 형식
- 명확하고 구체적인 답변 제공 (모호한 표현 지양)
- 정보를 **표/리스트**로 구조화

### AI 브리핑 최적화
- 네이버 AI 브리핑이 답변을 추출할 수 있도록 **정의형 문장** 작성
  - 예: "학교폭력예방법 제17조에 따른 전학 조치란, ~를 의미합니다."
- 첫 문단에 질문에 대한 **직접적 답변(Direct Answer)** 배치
- 법령 조항 번호와 함께 구체적 **수치/기한** 명시

### 구조화된 답변 포맷
- "~란 무엇인가?" → 정의 1~2문장 → 상세 설명 순서
- 절차 설명 시 반드시 번호 매기기 (1단계, 2단계...)
- 기한/비용/요건 등 핵심 정보는 표로 정리

### 초입문자 모드 AEO 전략

초입문자 콘텐츠에서는 AI 브리핑이 **행동 가이드형 답변**을 추출할 수 있는 구조를 본문에 자연스럽게 녹인다.

**답변 추출 구조 패턴:**
```
Q: [초입문자가 실제로 검색할 질문]
A: [직접 답변 1~2문장]. 구체적으로는 다음과 같습니다:
   1) [행동1], 2) [행동2], 3) [행동3], 4) [행동4], 5) [행동5]
```

**적용 예시:**
```
Q: 아이가 학폭 가해자로 통보받았을 때 해야 할 일은?
A: 학폭 가해자 통보를 받았을 때, 48시간 내에 해야 할 5가지는
   다음과 같습니다: 1) 아이의 이야기를 차분히 듣기, 2) 통보 문서
   확인, 3) 사실관계 기록, 4) 학교 담당자에게 절차 확인,
   5) 전문가 상담 여부 결정입니다.
```

**초입문자 AEO 체크포인트:**
- [ ] 파트2("해야 할 것")에 번호 리스트형 직접 답변이 포함되었는가?
- [ ] 서론 또는 파트2 도입부에 AI가 추출 가능한 요약 문장이 있는가?
- [ ] 질문형 H2 소제목이 초입문자의 실제 검색 질문과 일치하는가?

### AEO 체크포인트
- [ ] 질문형 소제목이 사용되었는가?
- [ ] 각 질문에 대한 직접적인 답변이 제공되는가?
- [ ] 표/리스트로 정보가 구조화되었는가?
- [ ] 모호한 표현 없이 구체적으로 작성되었는가?
- [ ] 정의형 문장이 포함되어 있는가? (AI 브리핑 대응)
- [ ] 첫 문단에 Direct Answer가 배치되어 있는가?

## GEO (Generative Engine Optimization)

생성형 AI가 콘텐츠를 이해하고 답변 소스로 활용할 수 있도록 최적화한다.

### GEO 핵심 전략
- 생성형 AI가 콘텐츠를 이해하고 **재구성할 수 있는 구조** 설계
- **팩트 기반**의 명확한 정보 제공 (모호한 표현 최소화)
- **출처를 명시**하여 AI가 신뢰도를 판단할 수 있도록 함
- 법령 조항 번호, 구체적 수치/기한을 포함한 정보 작성

### GEO 체크포인트
- [ ] 팩트 기반의 명확한 정보가 제공되는가?
- [ ] 출처(법령, 판례, 통계)가 명시되어 있는가?
- [ ] 구조화된 답변 포맷("~란?" → 정의 → 상세)이 적용되었는가?

## 메타태그 (해시태그) 구성 가이드

블로그 검색 최적화를 위해 게시글에 사용할 관련 해시태그(#) **15개 이상** 추천한다.

| 유형 | 개수 | 예시 |
|------|------|------|
| 핵심 키워드 태그 | 3-5개 | #교원징계, #학교폭력, #소청심사 |
| 관련 법률/제도 태그 | 3-5개 | #청탁금지법, #학폭예방법, #행정소송 |
| 대상/상황 태그 | 3-5개 | #학부모, #현직교사, #가해학생 |
| 일반 법률 태그 | 3-5개 | #교육법전문변호사, #행정심판, #법률상담 |
| 브랜드/지역 태그 | 2-3개 | #염용주변호사, #법률사무소이지스, #서울교육청 |

### 메타태그 작성 원칙
1. 핵심 키워드를 가장 앞에 배치
2. 검색량이 높은 일반적인 태그와 구체적인 태그를 혼합
3. 브랜드 태그는 항상 포함
4. 총 15~20개 범위 유지 (너무 많으면 스팸 판정)

## 알고리즘 최적화 요약표

| 알고리즘 | 핵심 전략 | 체크 포인트 |
|---------|----------|------------|
| **C-RANK** | 전문성 + 신뢰도 | 법령/판례 인용, 경력 자연스럽게 언급 |
| **D.I.A+** | 체류시간 증대 | 훅, 의외성 통계, 스토리텔링, 질문 유도 |
| **SEO** | 키워드 최적화 | 제목/서론 키워드, H2/H3 구조화, 20회 미만 |
| **AEO** | AI 답변 최적화 | Q&A 형식, 질문형 소제목, 표/리스트 구조화, AI 브리핑 대응 |
| **GEO** | 생성형 AI 소스 최적화 | 팩트 기반, 출처 명시, 구조화된 답변 포맷 |

## 출력 형식

SEO팀의 산출물은 다음 구조로 출력한다:

```
## SEO 최적화 결과

### 키워드 밀도 보고서
| 키워드 | 출현 횟수 | 판정 |
|--------|----------|------|
| [키워드1] | X회 | ✅ 적정 / ⚠️ 과다 |
| [키워드2] | X회 | ✅ 적정 / ⚠️ 과다 |

### C-RANK 최적화 확인
- [ ] 공신력 있는 출처 인용 여부
- [ ] 전문성 표현 적절성
- [ ] 키워드 자연스러운 배치
- [ ] 서론에 실무 경험/위촉·출강 경력이 주제와 연결되어 언급되었는가
- [ ] 본론에 '실무 포인트' 박스가 1개 이상 포함되었는가

### D.I.A+ 최적화 확인
- [ ] 훅(Hook) 존재 여부
- [ ] 스토리텔링 요소
- [ ] 질문 유도 포함

### AEO/GEO 최적화 확인
- [ ] 질문형 소제목
- [ ] 표/리스트 구조화
- [ ] 직접적 답변 형식
- [ ] AI 브리핑 대응 (정의형 문장, Direct Answer)
- [ ] GEO 대응 (팩트 기반, 출처 명시, 구조화된 답변)

### 최적화 본문
[SEO 최적화가 적용된 전체 본문]

### 메타태그 (15개+)
#태그1 #태그2 #태그3 ...
```
