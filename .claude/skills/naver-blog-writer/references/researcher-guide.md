# 리서치팀 가이드 (Researcher Guide)

## 필수 조사 항목

글 작성 전 WebSearch와 WebFetch를 활용하여 다음 정보를 반드시 조사한다:

| 조사 항목 | 출처 예시 | 목적 |
|----------|----------|------|
| 최신 법률/제도 변경사항 | 법제처, 교육부, 국가법령정보센터 | 정확성 확보 |
| 관련 대법원/행정법원 판례 | 대법원 종합법률정보 | 전문성 강화 |
| 교육부/교육청 공식 지침 | 교육부, 시도교육청 | 신뢰도 향상 |
| 관련 통계 데이터 | 정부 기관, 연구기관 | 의외성 제공 |
| 최근 사회적 이슈/뉴스 | 언론사 (최근 1년 이내) | 시의성 확보 |

---

## 공신력 있는 출처 목록

### 1순위: 정부/공공기관
- **법제처** (law.go.kr) — 법령 원문
- **국가법령정보센터** (law.go.kr) — 법률/시행령/시행규칙
- **교육부** (moe.go.kr) — 교육 정책/지침
- **시도교육청** — 지역별 교육 정책
- **대법원 종합법률정보** (glaw.scourt.go.kr) — 판례 검색
- **헌법재판소** — 헌법재판 결정

### 2순위: 연구/학술기관
- **한국교육개발원** (KEDI)
- **한국교육과정평가원** (KICE)
- **한국형사·법무정책연구원**
- **학술 논문** (KCI 등재지)

### 3순위: 언론
- **주요 일간지** — 조선, 중앙, 동아, 한겨레, 경향 등
- **교육 전문 매체** — 한국교육신문, 에듀인뉴스 등
- **법률 전문 매체** — 법률신문, 로앤비 등

## 조사 절차

### Step 1: 기획팀 요청사항 분석
- 기획팀 콘텐츠 플랜에서 **리서치 요청사항** 확인
- 메인/서브 키워드 파악
- 필수 조사 항목 목록화

### Step 2: 법률/제도 조사
- 관련 법률 조항 원문 확인 (최신 개정 여부 반드시 확인)
- 시행령/시행규칙까지 확인
- 최근 법률 개정사항 별도 정리

### Step 3: 판례 조사
- 대법원 판례 중심으로 조사
- 최근 3년 이내 판례 우선
- 판결 요지를 일반인이 이해할 수 있도록 요약
- WebFetch로 판례 전문 확인 후 핵심 판시사항 추출

### Step 4: 통계/뉴스 조사
- 최근 1년 이내 통계 데이터 수집
- 관련 최신 뉴스 3~5건 확인
- 사회적 이슈와의 연결고리 파악

### Step 5: 경쟁 콘텐츠 분석
- 해당 키워드 검색 결과 **상위 3~5개 블로그 글** 분석
  - 구조, 분량, 이미지 수 파악
- **정보 갭(Information Gap)** 발견 — 상위 글들이 다루지 않는 정보 식별
- **차별화 포인트** 1~2가지 도출
  - 예: "다른 글에서 다루지 않는 최신 판례 추가", "실제 절차 경험 기반 타임라인 제공"

### 조사 최소 기준

각 항목별 최소 조사 건수를 반드시 충족한다:

| 조사 항목 | 최소 기준 | 비고 |
|----------|----------|------|
| 관련 법률/제도 | **1건 이상** | 핵심 법률 조항 + 최신 개정 여부 |
| 판례 | **2건 이상** | 대법원/행정법원 판례 우선 |
| 통계 데이터 | **1건 이상** | 정부/연구기관 공식 통계 |
| 최신 뉴스 | **3건 이상** | 최근 1년 이내, 주요 언론사 |

※ 주제에 따라 특정 항목이 존재하지 않는 경우, 그 사유를 리서치 노트에 명시한다

## 주의사항

### 반드시 지켜야 할 원칙
1. **출처 명시** — 모든 정보에 출처(기관명, 날짜)를 반드시 기재
2. **최신성 확인** — 법률/제도는 최신 개정 여부를 반드시 확인 (구법 인용 금지)
3. **정확성 우선** — 불확실한 정보는 포함하지 않음
4. **익명화** — 판례/사례의 개인정보는 반드시 익명 처리
5. **편향 금지** — 한쪽 입장만 대변하지 않고 객관적 사실 중심

### 피해야 할 것
- 블로그/카페 등 비공식 출처 인용
- 날짜 불명확한 자료 사용
- 검증되지 않은 통계 수치
- 폐지/개정된 구법 조항 인용

## 출력 형식

리서치 노트는 다음 구조로 출력한다:

```
## 리서치 노트

### 1. 관련 법률/제도
- **[법률명]** 제X조 (시행일: YYYY.MM.DD)
  - 핵심 내용: [요약]
  - 최근 개정사항: [있으면 기재]
  - 출처: [URL 또는 기관명]

### 2. 주요 판례
- **[판례번호]** (YYYY.MM.DD 선고)
  - 사건 개요: [요약]
  - 판결 요지: [핵심 판단]
  - 활용 포인트: [블로그 글에서 어떻게 활용할지]
  - 출처: [URL]

### 3. 통계 데이터
- **[통계명]** (YYYY년 기준)
  - 핵심 수치: [데이터]
  - 의미: [해석]
  - 출처: [기관명, URL]

### 4. 최신 뉴스/이슈
- **[뉴스 제목]** (YYYY.MM.DD)
  - 핵심 내용: [요약]
  - 시의성 활용: [서론에서 어떻게 활용할지]
  - 출처: [언론사, URL]

### 5. 경쟁 콘텐츠 분석
- **분석 대상**: 상위 1~3위 블로그 글 URL
- **구조/분량**: [글자수, 이미지 수, 섹션 구성]
- **정보 갭**: [상위 글에서 빠진 정보]
- **차별화 포인트**: [우리 글의 차별화 전략]

### 6. 제작팀 활용 포인트
- 서론 활용: [시의성 있는 뉴스/통계로 훅 만들기]
- 본론1 활용: [법적 근거로 사용할 법률/판례]
- 본론2 활용: [절차/사례 설명에 쓸 판례/통계]
- 본론3 활용: [대응 전략의 근거가 될 자료]
- 결론 활용: [신뢰도를 높일 핵심 데이터]
```
