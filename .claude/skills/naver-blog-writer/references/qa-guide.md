# 검수팀 가이드 (QA Reviewer Guide)

> **⚠️ 검수 모델 지정**: 검수팀은 사실 정확성 검증의 신뢰도를 위해 **가장 최신·최상위 모델(claude-opus-4-6)**을 사용해야 한다. 하위 모델로 검수를 진행하지 않는다.

## 변호사 광고에 관한 규정 (2025.6.30 개정) 준수사항

**대한변호사협회 「변호사 광고에 관한 규정」 (2025.6.30 개정)**

### 규정 요약표

| 구분 | 내용 |
|------|------|
| **금지** | 허위/과장 광고, 비교 광고, 성과 보장, 최상급 표현, 공포 조장, 결과 보장("기각 시 100% 환불" 등), 수임료 할인 홍보 |
| **제한** | 승소율/실적 수치, 무료상담 강조, '전문'/'전담' 표현(제7조: 주로 취급하는 업무에 한해 사용 가능) |
| **AI 관련** | AI 사용 광고 시 협회 등록 필수, 검토 변호사 이름 기재 의무 |
| **허용** | 객관적 사실(경력, 학력, 전문분야), 일반적인 법률정보 제공 |

### 절대 금지 표현

| 유형 | 금지 예시 | 위반 사유 |
|------|-----------|----------|
| 승소율/성과 과장 | "승소율 95%", "00건 승소", "고객 만족도 1위" | 검증 불가, 과장 |
| 최상급 표현 | "최고의 변호사", "유일한 전문가", "가장 뛰어난" | 비교 광고 |
| 타 변호사 비하 | "다른 변호사와 다르게...", "타 사무소보다" | 비방 |
| 직접적 광고 | "무료 상담", "할인 이벤트", "특가", "지금 바로" | 과잉 광고 |
| 결과 보장 | "기각 시 100% 환불", "승소 보장", "결과 보장" | 결과 보장 금지 (2025 개정) |
| 수임료 할인 | "수임료 할인", "착수금 할인", "비용 할인 이벤트" | 수임료 할인 홍보 금지 (2025 개정) |
| 전문 표현 남용 | "OO 전문 변호사" (주로 취급하지 않는 분야) | 제7조 위반 — 주로 취급하는 업무에 한해 사용 가능 |
| 허위/과장 경력 | 검증 불가능한 경력, 과장된 수치 | 허위 광고 |
| 공포 조장 | "당장 대응하지 않으면 큰일납니다", "시간이 없습니다" | 공포 유발 |

### 권장 표현

- "교육법 분야에서 9년간 다양한 사건을 담당해 왔습니다"
- "서울시교육청 법무팀에서 5년간 근무한 경험을 바탕으로"
- "실질적인 도움을 드릴 수 있도록 최선을 다하겠습니다"
- "전문가의 조력이 필요하시다면 언제든 연락 주세요"
- "함께 최선의 해결 방안을 찾아보겠습니다"

## 품질 관리 체크리스트

### 1. 분량 확인

각 섹션별 **목표 범위**와 **허용 범위(±10%)**는 다음과 같다. 목표 범위 내이면 ✅, 허용 범위 내이면 ✅(허용), 허용 범위 초과 시 ⚠️로 판정한다.

**키워드 경쟁도별 총 분량 기준:**

| 키워드 경쟁도 | 권장 글자수 (공백 제외) | 허용 범위 (±10%) |
|-------------|---------------------|-----------------|
| 낮음 (월 검색량 500 미만) | 2,000~2,500자 | 1,800~2,750자 |
| 중간 (월 검색량 500~3,000) | 2,500~3,000자 | 2,250~3,300자 |
| 높음 (월 검색량 3,000 이상) | 3,000~4,000자 | 2,700~4,400자 |

**섹션별 기준 (중간 경쟁도 기본):**

| 섹션 | 목표 범위 | 허용 범위 (±10%) |
|------|----------|-----------------|
| 서론 | 250~300자 | 225~330자 |
| 본론1 | 500~600자 | 450~660자 |
| 본론2 | 500~600자 | 450~660자 |
| 본론3 | 500~600자 | 450~660자 |
| 결론 | 200~250자 | 180~275자 |
| **총합** | **2,500~3,000자** | **2,250~3,300자** |

※ 높음 경쟁도일 경우 각 섹션 분량도 비례하여 확대한다

- [ ] 각 섹션이 허용 범위 내에 있는가?
- [ ] 전체 글자 수가 허용 범위 내에 있는가?
- [ ] 문단당 3~5문장 유지

**글자 수 측정 기준:**
- **포함**: 순수 텍스트 (한글, 영문, 숫자, 문장부호)
- **제외**: 공백, 마크다운 태그(`##`, `**`, `- ` 등), 이미지 주석(`[이미지: ...]`), HTML 태그
- **측정 방법**: 각 섹션의 본문 텍스트만 추출하여 공백을 제거한 후 글자 수를 센다

### 2. SEO/알고리즘 최적화
- [ ] 키워드 밀도: 각 키워드 **20회 이하**
- [ ] H2/H3 태그로 구조화
- [ ] 질문형 소제목 사용 (AEO 최적화)
- [ ] 법령/판례/정부자료 출처 인용
- [ ] 제목 25자 이내
- [ ] 메타태그 15개 이상

### 3. 어조 및 표현
- [ ] 따뜻하고 친근한 어조 유지
- [ ] 전문용어 쉽게 풀어쓰기
- [ ] 서론에 해당 주제 관련 실무 경험 언급 (1~2문장)
- [ ] 위촉·출강 경력이 주제와 자연스럽게 연결 (해당 시)
- [ ] 경력 언급이 단순 나열이 아닌 주제 연결형인지 확인
- [ ] 독자 공감 문장 포함

### 4. 부가 요소
- [ ] CTA 문구 포함 (변호사 광고에 관한 규정 준수)
- [ ] 이미지 배치 위치 5곳 지정
- [ ] 이미지 프롬프트 5개 (영문)
- [ ] 제목 5개 추천
- [ ] 메타태그 15개 이상

### 5. 규정 준수
- [ ] 변호사 광고에 관한 규정 금지 표현 없음
- [ ] 공포 조장 표현 없음
- [ ] 과장/허위 표현 없음
- [ ] 승소율/성과 수치 없음
- [ ] 최상급 표현 없음
- [ ] 타 변호사 비하 없음
- [ ] 결과 보장 표현 없음 ("기각 시 100% 환불" 등)
- [ ] 수임료 할인 홍보 없음

### 6. 사실 정확성
- [ ] 인용된 법률 조항이 최신 법률과 일치
- [ ] 판례 번호/날짜 정확성 확인
- [ ] 통계 데이터 출처/날짜 확인
- [ ] 제도/절차 설명의 정확성

### 7. 오탈자/문법
- [ ] 오탈자 없음
- [ ] 맞춤법 확인
- [ ] 문장 부호 적절성
- [ ] 띄어쓰기 확인

### 8. 통합 최종 점검
- [ ] 내부 링크 2~3개 포함 확인
- [ ] 내부 링크가 `references/blog-posts.json`에 실제 존재하는 글인지 검증 (logNo 또는 URL 대조)
- [ ] 모바일 가독성 (문단 3~4줄 이내)
- [ ] 첫 문단 2~3줄 이내 핵심 답변 포함 (스마트블록/AEO 대응)
- [ ] AI 특유 표현 제거 확인 (기계적 반복 구조, "~에 대해 알아보겠습니다" 등)
- [ ] 변호사 광고에 관한 규정 (2025.6.30 개정) 준수
- [ ] 결과 보장 / 수임료 할인 표현 없음

### 9. 모드별 추가 점검 항목

#### 【초입문자 가이드 모드】 추가 점검

- [ ] 서론이 **상황묘사→감정인정→안심메시지→가이드예고** 순서를 따르는가?
- [ ] 본문이 **하지 말 것→해야 할 것→왜→다음 단계** 4파트 구조인가?
- [ ] 법률 용어가 처음 등장 시 **괄호 안에 쉬운 풀이**가 포함되어 있는가?
- [ ] 행동 가이드(해야 할 것/하지 말 것)가 구체적이고 즉시 실행 가능한가?
- [ ] CTA가 심리적 장벽 해소 중심인가? (예: "어디서부터 시작해야 할지 모르겠다면...")
- [ ] 하단에 **2단 연결 블록**("더 깊이 알고 싶다면" + "지금 상황이 복잡하다면")이 포함되어 있는가?
- [ ] 연결 블록의 링크가 해당 주제와 직접 관련된 전문 정보/성공 사례 글을 가리키는가?
- [ ] 키워드가 **초입문자 검색어**(롱테일, 감정형, 구어체)를 타깃하고 있는가?
- [ ] 파트2에 AI가 추출 가능한 **번호 리스트형 직접 답변** 구조가 포함되어 있는가?
- [ ] 제목에 "처음", "첫 대응", "어떻게" 등 초보자 시그널 키워드가 포함되어 있는가?

#### 【성공 사례 모드】 추가 점검

- [ ] 5파트 구조(**상황설명→법적쟁점→대응전략→결과→시사점**)를 준수하는가?
- [ ] 승소율, 성과 수치 등 **구체적 수치 표현이 없는가?** (예: "승소율 95%", "00건 승소" 금지)
- [ ] 의뢰인 정보가 **완전히 익명화**되었는가? (이름, 학교명, 지역 등 모두 제거)
- [ ] 결과 보장 표현이 없는가? (예: "반드시 승소", "100% 구제" 금지)
- [ ] 비교/비방 표현이 없는가? (예: "다른 변호사에게 갔다면..." 금지)
- [ ] 톤이 **사실 기반, 차분한 어조**인가? (감정적 과장, 자기 자랑 없음)
- [ ] 시사점이 독자에게 적용 가능한 교훈 중심으로 작성되었는가?
