# 검수팀 가이드 (QA Reviewer Guide)

## 변호사업무광고규정 준수사항

**대한변호사협회 변호사업무광고에관한규정 제3조~제5조**

### 규정 요약표

| 구분 | 내용 |
|------|------|
| **금지** | 허위/과장 광고, 비교 광고, 성과 보장, 최상급 표현, 공포 조장 |
| **제한** | 승소율/실적 수치, 무료상담 강조, 보수 할인 광고 |
| **허용** | 객관적 사실(경력, 학력, 전문분야), 일반적인 법률정보 제공 |

### 절대 금지 표현

| 유형 | 금지 예시 | 위반 사유 |
|------|-----------|----------|
| 승소율/성과 과장 | "승소율 95%", "00건 승소", "고객 만족도 1위" | 검증 불가, 과장 |
| 최상급 표현 | "최고의 변호사", "유일한 전문가", "가장 뛰어난" | 비교 광고 |
| 타 변호사 비하 | "다른 변호사와 다르게...", "타 사무소보다" | 비방 |
| 직접적 광고 | "무료 상담", "할인 이벤트", "특가", "지금 바로" | 과잉 광고 |
| 허위/과장 경력 | 검증 불가능한 경력, 과장된 수치 | 허위 광고 |
| 공포 조장 | "당장 대응하지 않으면 큰일납니다", "시간이 없습니다" | 공포 유발 |

### 권장 표현

- "교육법 분야에서 9년간 다양한 사건을 담당해 왔습니다"
- "서울시교육청 법무팀에서 5년간 근무한 경험을 바탕으로"
- "실질적인 도움을 드릴 수 있도록 최선을 다하겠습니다"
- "전문가의 조력이 필요하시다면 언제든 연락 주세요"
- "함께 최선의 해결 방안을 찾아보겠습니다"

## 품질 관리 체크리스트

### 1. 분량 확인

각 섹션별 **목표 범위**와 **허용 범위(±10%)**는 다음과 같다. 목표 범위 내이면 ✅, 허용 범위 내이면 ✅(허용), 허용 범위 초과 시 ⚠️로 판정한다.

| 섹션 | 목표 범위 | 허용 범위 (±10%) |
|------|----------|-----------------|
| 서론 | 250~300자 | 225~330자 |
| 본론1 | 500~600자 | 450~660자 |
| 본론2 | 500~600자 | 450~660자 |
| 본론3 | 500~600자 | 450~660자 |
| 결론 | 200~250자 | 180~275자 |
| **총합** | **2,500~3,000자** | **2,250~3,300자** |

- [ ] 각 섹션이 허용 범위 내에 있는가?
- [ ] 전체 글자 수가 허용 범위 내에 있는가?
- [ ] 문단당 3~5문장 유지

**글자 수 측정 기준:**
- **포함**: 순수 텍스트 (한글, 영문, 숫자, 문장부호)
- **제외**: 공백, 마크다운 태그(`##`, `**`, `- ` 등), 이미지 주석(`[이미지: ...]`), HTML 태그
- **측정 방법**: 각 섹션의 본문 텍스트만 추출하여 공백을 제거한 후 글자 수를 센다

### 2. SEO/알고리즘 최적화
- [ ] 키워드 밀도: 각 키워드 **20회 이하**
- [ ] H2/H3 태그로 구조화
- [ ] 질문형 소제목 사용 (AEO 최적화)
- [ ] 법령/판례/정부자료 출처 인용
- [ ] 제목 25자 이내
- [ ] 메타태그 15개 이상

### 3. 어조 및 표현
- [ ] 따뜻하고 친근한 어조 유지
- [ ] 전문용어 쉽게 풀어쓰기
- [ ] 경력 및 전문성 자연스럽게 언급
- [ ] 독자 공감 문장 포함

### 4. 부가 요소
- [ ] CTA 문구 포함 (변호사광고규정 준수)
- [ ] 이미지 배치 위치 5곳 지정
- [ ] 이미지 프롬프트 5개 (영문)
- [ ] 제목 5개 추천
- [ ] 메타태그 15개 이상

### 5. 규정 준수
- [ ] 변호사업무광고규정 금지 표현 없음
- [ ] 공포 조장 표현 없음
- [ ] 과장/허위 표현 없음
- [ ] 승소율/성과 수치 없음
- [ ] 최상급 표현 없음
- [ ] 타 변호사 비하 없음

### 6. 사실 정확성
- [ ] 인용된 법률 조항이 최신 법률과 일치
- [ ] 판례 번호/날짜 정확성 확인
- [ ] 통계 데이터 출처/날짜 확인
- [ ] 제도/절차 설명의 정확성

### 7. 오탈자/문법
- [ ] 오탈자 없음
- [ ] 맞춤법 확인
- [ ] 문장 부호 적절성
- [ ] 띄어쓰기 확인

## QA 보고서 출력 형식

검수팀의 QA 보고서는 다음 구조로 출력한다:

```
## QA 보고서

### 1. 분량 검수
| 섹션 | 글자 수 (공백 제외) | 목표 | 허용 범위 (±10%) | 판정 |
|------|-------------------|------|-----------------|------|
| 서론 | X자 | 250~300자 | 225~330자 | ✅ / ⚠️ |
| 본론1 | X자 | 500~600자 | 450~660자 | ✅ / ⚠️ |
| 본론2 | X자 | 500~600자 | 450~660자 | ✅ / ⚠️ |
| 본론3 | X자 | 500~600자 | 450~660자 | ✅ / ⚠️ |
| 결론 | X자 | 200~250자 | 180~275자 | ✅ / ⚠️ |
| **총합** | **X자** | **2,500~3,000자** | **2,250~3,300자** | ✅ / ⚠️ |

### 2. 광고규정 검수
- 금지 표현 발견: 없음 ✅ / [발견된 표현 목록]
- 공포 조장 표현: 없음 ✅ / [발견된 표현 목록]
- 과장/허위 표현: 없음 ✅ / [발견된 표현 목록]

### 3. 오탈자/문법 검수
- 발견된 오탈자: 없음 ✅ / [오탈자 목록 및 수정 제안]
- 맞춤법 오류: 없음 ✅ / [오류 목록]

### 4. 사실 정확성 검수
- 법률 인용 정확성: ✅ / ⚠️ [문제점]
- 판례 정확성: ✅ / ⚠️ [문제점]
- 통계 정확성: ✅ / ⚠️ [문제점]

### 5. 부가 요소 확인
- 제목 5개: ✅ / ⚠️
- 이미지 프롬프트 5개: ✅ / ⚠️
- 메타태그 15개+: ✅ (X개) / ⚠️ (X개, 부족)

### 6. 종합 판정
- **최종 승인**: ✅ 승인 / ⚠️ 수정 필요
- **수정 필요 사항**: [있을 경우 구체적으로 기재]

### 최종 승인 본문
[검수 완료된 최종 본문 - 수정사항이 있으면 반영된 버전]
```
