# 법률 초입문자 포착 콘텐츠 기획서

> "어떻게 해야 하는지 몰라서" 법률서비스를 이용하지 못하는 39.6%를 위한 콘텐츠 전략

---

## 1. 왜 이 콘텐츠가 필요한가

### 데이터 근거

KDI가 케이스탯 리서치에 의뢰한 설문조사(리걸타임즈, 2024.6) 결과:

```
법률서비스 미충족 수요 원인:

  ■■■■■■■■■■■■■■■■■■■■  39.6%  "어떻게 해야 하는지 몰라서"
  ■■■■■■■■■■■■■■■■      31.4%  "비용이 부담돼서"
  ■■■■■■                  12.1%  "법적 절차가 무섭고 스트레스"
  ■■■■■                   16.9%  기타
```

특히 교육법 분야는 이 경향이 더 심합니다:
- 학부모: 학폭 통지서를 처음 받았을 때, 변호사가 필요한 건지조차 모름
- 교원: 징계 절차를 처음 겪으면서, 소청심사라는 제도 자체를 모름
- 학교장: 법적 분쟁이 발생했을 때, 개인 책임 범위를 모름

### 현재 블로그의 한계

블로그 113편 중 **"법을 처음 접하는 사람"을 위한 안내 콘텐츠는 0편**입니다.

기존 콘텐츠는 이미 문제를 인식하고 해결 방법을 찾는 사람을 위해 설계되어 있습니다. "학교폭력 사안처리 절차의 이해"는 절차를 알고 싶은 사람에겐 훌륭하지만, **"이게 법적 문제인지조차 모르는 사람"**에게는 닿지 않습니다.

### 비유

```
현재 블로그:     병원에서 "수술 절차 안내서"를 제공하는 것
필요한 콘텐츠:   "이 증상이 병원에 가야 하는 건지" 알려주는 것
```

## 2. 타겟 페르소나 정의

### 페르소나 A: "갑자기 학부모" (학교폭력 관련)

```
이름: 김지현 (가명, 42세, 직장인)
상황: 아이가 학교에서 학폭위 가해자로 통지서를 받아 옴
심리: 패닉, 분노, 혼란, 죄책감이 뒤섞인 상태
지식: "학폭위"가 뭔지 대충 알지만, 법적 절차라는 인식이 없음
검색: "아이가 학폭 가해자 어떡하죠" "학폭위 통보 받으면"
핵심 니즈: 지금 당장 뭘 해야 하는지, 변호사가 필요한 건지
```

### 페르소나 B: "당황한 교사" (교원 징계 관련)

```
이름: 박성호 (가명, 38세, 중학교 교사 8년차)
상황: 학부모 민원으로 아동학대 고소당함 / 사교육카르텔 수사 연락 받음
심리: 억울함, 두려움, 고립감
지식: 교원 징계 절차를 정확히 모름, 소청심사 존재 자체를 모를 수 있음
검색: "교사 고소당하면 어떻게" "교원 징계 받으면" "사교육카르텔 연루 교사"
핵심 니즈: 나에게 무슨 일이 일어나는 건지, 어떤 선택지가 있는지
```

### 페르소나 C: "걱정되는 학부모" (학교안전사고)

```
이름: 이수진 (가명, 45세, 전업주부)
상황: 아이가 학교에서 다쳤는데 학교 측의 대응이 미흡하다고 느낌
심리: 걱정, 억울함, "법적으로 뭔가 할 수 있나?" 궁금
지식: 학교안전공제회 존재조차 모름
검색: "학교에서 아이 다쳤을 때" "학교 사고 책임" "학교에서 다친 치료비"
핵심 니즈: 이게 법적 문제인지, 보상받을 수 있는지, 어디서부터 시작하는지
```

### 페르소나 D: "처음인 학교장" (학교 법적 분쟁)

```
이름: 정미영 (가명, 55세, 초등학교 교장 1년차)

심리: 불안, 책임감에 대한 무게
지식: 교육행정은 알지만 법적 절차에 대한 경험 없음
검색: "학교장 법적 책임" "학교 소송 당하면" "교장 개인 책임"
핵심 니즈: 내 책임 범위, 무엇을 해야 하고 무엇을 하면 안 되는지
```

## 3. 시리즈 설계: "처음이신가요?" 시리즈

### 시리즈 컨셉

**시리즈명:** 처음이신가요? — 교육법 첫걸음 가이드

**핵심 원칙:**
- 법률 용어를 쓰지 않고 상황을 먼저 설명
- 독자의 감정 상태를 인정하고 공감하는 것에서 시작
- "지금 당장 해야 할 것"을 먼저 알려주고, 법적 배경은 나중에
- 각 글의 마지막에 "더 깊이 알고 싶다면" 기존 전문 시리즈로 연결

**톤 앤 매너:**
```
❌ "학교폭력예방법 제17조에 의거하여..."
✅ "갑자기 이런 통보를 받으시면 당연히 놀라실 수밖에 없습니다.
    지금부터 차근차근 알려드릴게요."
```

2. 아이를 몰아세워 추궁하기
3. 학교에 항의 전화하기

(체크리스트 형태로 명확하게)
□ 1. 아이의 이야기를 차분히 듣기 (어떻게 들어야 하는지 구체적으로)
□ 2. 통보 문서 꼼꼼히 확인하기 (어떤 부분을 봐야 하는지)
□ 3. 사실관계 기록 시작하기 (무엇을, 어떤 형식으로)
□ 4. 학교 담당자에게 절차 확인 전화하기 (뭘 물어봐야 하는지)
□ 5. 전문가 상담 여부 결정하기 (판단 기준 제시)

## 🚨 지금 가장 중요한 것: 72시간 골든타임
(의외성 + 긴박감으로 체류시간 확보)

"고소를 당한 후 첫 72시간이 사건의 방향을 결정합니다."

### 첫 24시간: 절대 하지 말아야 할 것
- ❌ 고소한 학부모에게 연락하기
- ❌ 동료 교사들에게 사건 내용 상세히 이야기하기
- ❌ SNS에 억울함 호소하기
- ❌ 혼자서 경찰 조사에 응하기

### 첫 24시간: 반드시 해야 할 것
- ✅ 해당 수업/지도 상황을 최대한 상세히 기록

- ✅ 당시 주변에 있던 증인(동료 교사, 학생) 파악
- ✅ 교원단체 또는 전문 변호사에게 즉시 연락

### 24~72시간: 방어 전략 기초 세우기
- 교원지위법에 따른 교육활동 보호 조치 확인
- 경찰 조사 일정 확인 및 변호사 동행 여부 결정
- 학교(교장, 교감)에 상황 공유 범위 결정

| 회차 | 제목 (안) | 핵심 질문 | 공략 키워드 |
|------|---------|----------|-----------|
| D-1 | 변호사 상담이 처음이신가요? 준비부터 진행까지 완벽 안내 | 상담 전 불안 해소 | 변호사 상담 처음, 법률상담 준비 |
| D-2 | 교육법 전문 변호사, 어떤 기준으로 골라야 하나요? | 선택 기준 (솔직한 가이드) | 교육법 변호사 선택, 학폭 변호사 |
| D-3 | 변호사 비용, 얼마나 들고 어떤 구조인가요? | 비용 구조 투명 설명 | 변호사 비용, 변호사 수임료 |
| D-4 | 변호사에게 사건을 맡기면 어떻게 진행되나요? | 수임 후 진행 과정 | 변호사 선임 후, 사건 진행 절차 |

**D-1 상세 구성안:**

```markdown
# 변호사 상담이 처음이신가요? 준비부터 진행까지 완벽 안내

## 서론
"변호사 사무실 문을 처음 두드리는 건 누구에게나 떨리는 일입니다.
'무슨 말을 해야 하지?', '바보 같은 질문 하면 어떡하지?',
'비용이 갑자기 청구되면?'... 이런 걱정, 충분히 이해합니다.
오늘은 상담 예약부터 상담실을 나오기까지,
처음 경험하시는 분의 눈높이에서 안내해 드리겠습니다."

## 상담 전: 이것만 준비하시면 됩니다

### 가져오시면 좋은 것
| 사건 유형 | 준비물 |
|---------|-------|
| 학교폭력 | 학폭위 통지서, 학교 공문, 관련 메시지 캡처 |
| 교원 징계 | 징계 통보서, 소명서 사본, 관련 서류 |
| 학교안전사고 | 사고 경위서, 진단서, 치료비 영수증 |
| 잘 모르겠다면 | 아무것도 안 가져오셔도 괜찮습니다 |

### 사전에 정리하면 좋은 것 (메모 형태도 OK)
- 사건이 일어난 날짜와 경위 (기억나는 대로)
- 현재 진행 상황 (어디까지 진행되었는지)
- 가장 궁금한 점 1~2가지

💡 "완벽하게 준비하실 필요 없습니다.
     상담은 '정리되지 않은 상태'에서 시작해도 괜찮습니다.
     정리를 도와드리는 것도 변호사의 역할이니까요."

## 상담 당일: 이렇게 진행됩니다

### STEP 1. 사건 이야기 (15~20분)
- 편하게 상황을 설명해 주시면 됩니다
- 변호사가 중간중간 질문을 드릴 수 있습니다
- 감정적이 되셔도 괜찮습니다 (누구나 그렇습니다)

### STEP 2. 법적 분석 (10~15분)
- 현재 상황의 법적 의미를 설명드립니다
- 어떤 선택지가 있는지 안내합니다
- 모르는 용어가 있으면 바로 질문하세요

### STEP 3. 방향 논의 (10~15분)
- 대응 방안별 장단점 설명
- 예상 기간, 절차 안내
- 비용에 대한 안내

### STEP 4. 마무리
- 바로 결정하실 필요 없습니다
- 집에 돌아가서 가족과 상의하셔도 됩니다
- "오늘 상담만으로 충분하다"고 판단되면 그것으로 끝입니다

## 상담 비용에 대해

"상담만 받고 사건을 맡기지 않아도 됩니다.
상담은 '정보를 얻는 과정'이지, '계약하는 자리'가 아닙니다."

- 상담 비용은 사무소마다 다르지만, 사전에 안내받으실 수 있습니다
- "상담 후 반드시 사건을 맡겨야 한다"는 규칙은 없습니다
- 상담만으로 방향이 정리되는 경우도 많습니다

## 자주 하시는 질문

Q. "제 사건이 너무 사소한 것 같아서 변호사한테 가기 민망해요"
→ 사소해 보이는 문제가 법적으로 큰 의미를 가지는 경우가 많습니다.
   판단은 전문가에게 맡기시는 것이 현명합니다.

Q. "변호사 상담 가면 바로 비용을 내야 하나요?"
→ 사전에 상담 비용 안내를 받으시며, 그 외에 추가로 청구되는 비용은 없습니다.

Q. "상담만 받고 다른 변호사에게 맡겨도 되나요?"
→ 네, 전혀 문제 없습니다. 여러 변호사와 상담 후 결정하시는 것도
   좋은 방법입니다.

## 마치며
"법률 문제 앞에서 막막하신 마음, 잘 알고 있습니다.
상담은 그 막막함에서 한 발짝 나오는 과정입니다.
교육법 분야에서 9년간 많은 분들의 첫 상담을 함께했습니다.
어려운 상황에서 함께 방법을 찾아드리겠습니다."
```

## 4. 글 작성 원칙: 기존 프롬프트와 다른 점

초입문자 콘텐츠는 기존 법률 정보 글과 근본적으로 다른 접근이 필요합니다.

### 4-1. 제목 작성 원칙

```
기존 법률 정보 글 제목:
  "학교폭력 사안처리 절차의 이해(1) : 학교폭력 사안처리 흐름 완벽 가이드"
  → 이미 "사안처리 절차"를 알고 있는 사람이 검색하는 제목

초입문자 콘텐츠 제목:
  "아이가 학폭 가해자로 통보받았습니다. 지금 뭘 해야 하나요?"
  → 방금 그 상황에 처한 사람이 검색하는 제목
```

**원칙:**
- 법률 용어가 아닌 **상황 묘사**로 시작
- **질문형** 또는 **1인칭 고백형** 사용
- 감정이 담긴 검색어에 대응 ("어떡해야", "어떻게", "도와주세요")

### 4-2. 서론 작성 원칙

```
기존 법률 정보 글 서론:
  "안녕하세요, 오늘도 여러분의 법률 고민을 함께 나누는 염용주 변호사입니다.
   최근 학교폭력 사건 처리가 교육지원청으로 이관되면서..."
  → 정보 제공 위주

초입문자 콘텐츠 서론:
  "갑자기 학교에서 전화가 왔습니다.
   심장이 내려앉는 기분이셨을 겁니다.
   먼저 말씀드릴게요. 이 상황이 아이의 인생을 망치는 건 아닙니다.
   지금부터 하나씩 알려드리겠습니다."
  → 감정 공감 → 안심 → 가이드 예고
```

**공식:**
```
[1] 상황 묘사 (독자가 "이거 나 얘기다!" 느끼게)
[2] 감정 인정 (놀라셨죠, 당연합니다)
[3] 안심 메시지 (하지만 ~입니다)
[4] 가이드 예고 (지금부터 하나씩 알려드리겠습니다)
```

### 4-3. 본문 구성 원칙

```
기존 글:  개념 → 법적 근거 → 절차 → 대응 전략
초입문자:  하지 말 것 → 해야 할 것 → 왜 그런지 → 다음 단계
```

**핵심 차이점:**
- **"하지 말 것"을 먼저** — 초입문자는 패닉 상태에서 실수를 저지르기 쉬움
- **행동 가이드가 법적 설명보다 먼저** — "왜"보다 "뭘"이 급함
- **체크리스트 형태** 적극 활용 — 하나씩 따라 할 수 있도록
- **법률 용어 사용 시 반드시 괄호 풀이** — "소청심사(쉽게 말해, 징계에 불복하는 절차)"

### 4-4. CTA 원칙

```
기존 CTA:
  "전문가의 조력을 구하시기 바랍니다"

초입문자 CTA:
  "지금 상황이 어디쯤인지 모르시겠다면,
   전화 한 통으로 방향을 잡아드릴 수 있습니다.
   상담 자체가 부담이 되실까 걱정하지 마세요."
```

## 6. SEO / 스마트블록 / AEO 최적화 전략

### 6-1. 검색어 매핑

초입문자가 실제로 검색하는 키워드는 **롱테일(long-tail)이며 감정적**입니다.

| 기존 콘텐츠가 잡는 키워드 | 초입문자 콘텐츠가 잡아야 할 키워드 |
|----------------------|------------------------------|
| 학교폭력 사안처리 절차 | 아이 학폭 가해자 어떡해 |
| 교원 징계 절차 가이드 | 교사 고소당하면 어떻게 |
| 학교안전사고 법적 책임 | 학교에서 아이 다쳤을 때 |
| 교원소청심사 기초 | 징계 불복 방법 |
| 사교육카르텔 대응 전략 | 사교육카르텔 연루 교사 어떡해 |

### 6-2. 스마트블록 최적화

네이버 스마트블록은 **검색 의도별로 결과를 분류**합니다. 초입문자 콘텐츠는 "기본 정보" 또는 "첫 대응" 블록에 노출되도록 설계해야 합니다.

- 제목에 "처음", "첫 대응", "어떻게"를 포함
- 첫 2~3문장에 직접적 답변 배치 (스마트블록 미리보기에 노출)
- H2 소제목을 질문형으로 ("지금 뭘 해야 하나요?", "비용은 얼마인가요?")

### 6-3. AEO 최적화

AI 브리핑이 답변으로 추출할 수 있는 구조:

```
Q: 아이가 학폭 가해자로 통보받았을 때 해야 할 일은?
A: 학폭 가해자 통보를 받았을 때, 48시간 내에 해야 할 5가지는
   다음과 같습니다: 1) 아이의 이야기를 차분히 듣기, 2) 통보 문서
   확인, 3) 사실관계 기록, 4) 학교 담당자에게 절차 확인,
   5) 전문가 상담 여부 결정입니다.
```

이런 구조가 글 안에 자연스럽게 녹아 있어야 합니다.

## 8. 성과 측정 기준

### 8-1. 초입문자 콘텐츠 KPI

| 지표 | 목표 | 측정 방법 |

| 검색 유입률 | 기존 글 대비 30% 이상 | 네이버 블로그 통계 |
| 평균 체류시간 | 3분 이상 | 블로그 통계 |
| 내부 링크 클릭률 | 전문 시리즈로 20% 이상 이동 | 링크 클릭 추적 |
| 상담 문의 전환 | 월 5건 이상 (초입문자 글 경유) | 상담 시 유입 경로 확인 |
| 공감/댓글 | 글당 평균 5개 이상 | 블로그 통계 |

### 8-2. A/B 테스트 제안

같은 주제에 대해 기존 스타일과 초입문자 스타일을 비교:

```
A안 (기존): "교원 징계 절차의 이해 — 징계위원회 출석 가이드"
B안 (초입문자): "징계위원회에 출석하라는 통보를 받았습니다. 어떡해야 하나요?"

→ 2주간 발행 후 조회수, 체류시간, 댓글 비교
```

## 글 유형 선택 (글 작성 전 결정)

### A. 전문 정보 글 (기본)

- 구조: 개념 → 법적 근거 → 절차 → 대응 전략
- 톤: 전문적이면서 친근한 어조

### B. 초입문자 가이드 (NEW)
- 대상: 법적 문제라는 인식조차 없거나, 처음 겪는 상황에서 막막한 사람

- 톤: 공감 우선, 안심 → 가이드 순서
- 제목: 상황 묘사형, 1인칭/질문형
- 서론: 감정 공감 → 안심 → 가이드 예고
- CTA: 심리적 장벽 해소 중심
- 연결: 글 하단에 관련 전문 시리즈 + 성공 사례 링크

### C. 성공 사례 (NEW)
- 대상: 비교 단계의 잠재 의뢰인
- 구조: 상황 → 쟁점 → 전략 → 결과 → 시사점
