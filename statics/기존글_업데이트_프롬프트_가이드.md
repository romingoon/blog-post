# 기존 글 업데이트 모드 — 프롬프트 가이드 전략

> 염용주 대표변호사의 교육법 쌀롱 블로그 기존 113편 최적화를 위한 실전 가이드
> 작성일: 2026년 2월

---

## 개요: 왜 기존 글 업데이트가 필요한가

현재 블로그의 113편은 대부분 **전통적 SEO(제목 키워드 + 해시태그) 중심**으로 작성되어 있습니다. 그러나 네이버 검색 환경은 2024~2026년 사이 근본적으로 변화했습니다.

| 변화 | 영향 | 기존 글 상태 |
|------|------|-------------|
| VIEW 탭 폐지 → 스마트블록 전환 | 의도별 블록 노출, 첫 문단 미리보기 | 대부분 도입부가 배경 설명형 |
| AI 브리핑 도입 (AEO) | 제로클릭 검색 60%+ 증가 | 정의형 문장·Direct Answer 부재 |
| 생성형 AI 검색 확산 (GEO) | AI가 콘텐츠를 재구성하여 답변 제공 | 구조화된 팩트 제시 부족 |
| D.I.A+ 알고리즘 강화 | 체류시간·스크롤 깊이가 순위에 직결 | 이탈 방지 장치(훅·시각 요소) 부족 |
| C-RANK 전문성 가중 상향 (40%) | 주제 일관성·전문성 신호 중요 | 일부 글에서 주제 확산 경향 |

**핵심 원칙:** 새 글을 쓰는 것이 아니라, 기존 자산의 가치를 극대화하는 것입니다. 이미 인덱싱된 URL과 축적된 C-RANK 점수를 보존하면서 내부를 업그레이드합니다.

### 1-1. 4사분면 우선순위 모델

모든 113편을 한꺼번에 업데이트할 수 없습니다. 아래 기준으로 우선순위를 매깁니다.

```
                    조회수 높음 (300+)
                         │
          ┌──────────────┼──────────────┐
          │   A. 즉시 업데이트     │   B. 최우선 업데이트   │
          │   (고성과-최신)     │   (고성과-구버전)    │
          │   → 리텐션+AEO 강화  │   → 풀 업데이트       │
  최신 ───┤──────────────┼──────────────┤─── 6개월+
  (6개월  │   C. 기회 발굴       │   D. 통합/재작성 검토  │
   이내)  │   (저성과-최신)     │   (저성과-구버전)    │
          │   → 제목+도입부 개선  │   → 시리즈 통합 또는   │

          └──────────────┼──────────────┘
                         │
                    조회수 낮음 (<100)
```

## PART 2. 업데이트 모드 프롬프트 — 본문

아래는 기존 블로그 글 작성 프롬프트에 **"업데이트 모드"** 섹션으로 추가할 내용입니다. AI 글작성 도구에 프롬프트로 입력할 때 사용합니다.

---

### [프롬프트 추가 섹션] 업데이트 모드 활성화

```markdown
## 🔄 업데이트 모드 (기존 글 최적화 시 사용)

### 모드 활성화 조건
- 이미 발행된 블로그 글을 수정할 때 이 모드를 사용합니다
- 새 글 작성 시에는 이 섹션을 무시하세요
- 아래 단계를 순서대로 적용하되, 글의 현재 상태에 따라 필요한 단계만 선택 가능

### 업데이트 원칙
1. **URL 보존**: 기존 포스트를 수정합니다. 새 글로 재발행하지 않습니다.
2. **핵심 키워드 유지**: 기존에 잡힌 검색 키워드는 변경하지 않습니다.
3. **추가 > 삭제**: 기존 내용을 삭제하기보다 보강·재구성합니다.
4. **최신화 표시**: 수정된 글 상단에 "📌 2026년 X월 업데이트" 뱃지를 추가합니다.
```

### [프롬프트 추가 섹션] STEP 1 — 도입부 리엔지니어링 (스마트블록 + AEO)

```markdown
## STEP 1. 도입부 리엔지니어링

### 현재 문제
기존 글 대부분이 아래 패턴입니다:
  "안녕하세요, 염용주 변호사입니다. 오늘은 ~에 대해 알아보겠습니다."
  → 스마트블록 미리보기에 핵심 정보가 노출되지 않음
  → AI 브리핑이 답변을 추출할 수 없음
  → 인사말로 3줄을 소진한 뒤에야 본론이 시작됨

### 업데이트 방법: 3단 도입부 구조

기존의 긴 인사말을 아래 **3단 구조**로 교체합니다.
기존 내용을 삭제하는 것이 아니라, 같은 분량을 더 효율적으로 재구성하는 것입니다.

---

**[3단 도입부 구조]**

┌─ ① 마이크로 인사 + Direct Answer (1~3줄)
│    인사 1문장 + 정의형 답변 + 핵심 수치/법령 → 한 호흡으로
│
├─ ② 공감 + 손실회피 (1~2줄)
│    독자 감정 인정 → 방치 시 불이익 암시
│
└─ ③ 글 안내 + 목차
     → "아래에서 ~를 다룹니다" + 글의 전체 목차(TOC)
     → 목차 직후 바로 본론 진입

---

**① 마이크로 인사 + Direct Answer**

기존처럼 인사말과 본론을 분리하지 않고, 한 문단으로 통합합니다.
이렇게 하면 스마트블록 미리보기 3줄 안에 인사와 핵심 답변이 모두 담깁니다.

> 안녕하세요, 교육법 전문 변호사 염용주입니다.
> {글의 핵심 질문에 대한 정의형 답변 1~2문장}.
> {관련 법령 조항 + 핵심 수치/기한}.

작성 규칙:
- "안녕하세요" 인사는 1문장(15자 이내)으로 압축
- 인사 직후, 같은 문단에서 바로 정의형 답변으로 이어짐
- "~란, ~입니다/~됩니다" 종결형 사용 (AI 브리핑 추출 최적)
- 법령 조항 번호, 구체적 수치(기한, 인원, 금액)를 반드시 포함
- 첫 3줄(120자 이내)에 핵심 답변이 완결적으로 제시되어야 함

**② 공감 + 손실회피**

핵심 답변 직후, 1~2문장으로 독자의 감정을 인정하고
"지금 대응하지 않으면 어떤 불이익이 있는지"를 짧게 암시합니다.

> {독자 감정 인정 문장}. {방치/지연 시 불이익 1가지 암시}.

작성 규칙:
- 독자가 이 글을 검색한 심리 상태를 반영 (불안, 당황, 분노, 막막함 등)
- 감정을 먼저 인정한 뒤("갑작스러운 통보에 놀라셨을 겁니다")
- 행동하지 않을 때의 손실을 한 줄로 암시("초기 대응을 놓치면 징계가 확정될 수 있습니다")
- 과장이나 공포 조성이 아닌, 실무 기반의 사실적 경고여야 함
- 변호사 광고 규정 준수: 성과 보장 표현 금지, 사실에 기반한 주의 환기만 허용

손실회피 패턴 예시:
| 카테고리 | 감정 인정 | 손실 암시 |
|---------|---------|---------|
| 교원 징계 | "갑작스러운 징계 통보에 당혹스러우실 겁니다" | "소명 기회 전 준비가 결과를 좌우합니다" |
| 학교폭력 | "아이 일이라 더 마음이 급하실 겁니다" | "학폭위 개최 전 72시간이 가장 중요한 대응 시간입니다" |
| 학교안전사고 | "아이가 다쳤다는 소식에 정신이 없으셨을 겁니다" | "초기 증거 확보를 놓치면 보상 청구가 어려워질 수 있습니다" |
| 행정소송 | "부당하다는 생각에 답답하실 겁니다" | "제소 기한(90일)을 넘기면 다툴 수 없게 됩니다" |

**③ 글 안내 + 목차 → 본론 진입**

공감/손실회피 직후, 글의 구조를 안내하고 **목차(TOC)를 배치**합니다.
목차는 글의 최상단(썸네일 바로 아래, ①② 직후)에 위치하므로
독자가 원하는 섹션으로 바로 이동할 수 있고, 체류시간이 증가합니다.

> 아래에서 ①{세부 주제 1} ②{세부 주제 2} ③{세부 주제 3}를
> 실무 경험을 바탕으로 정리해 드리겠습니다.
>
> **목차**
> 1. {세부 주제 1}
> 2. {세부 주제 2}
> 3. {세부 주제 3}

작성 규칙:
- 목차는 ②(공감+손실회피) 바로 다음, 본론 시작 직전에 배치
- 목차 항목은 클릭/스크롤 이동 가능한 앵커로 구성 (네이버 에디터 기능 활용)
- 3~5개 항목이 적정 (너무 길면 오히려 이탈 유발)
- 목차 직후 바로 본론 첫 번째 섹션이 시작됨 (추가 인사말 없음)

### 전체 예시 — "사립학교 교원징계위원회" 업데이트

**[BEFORE — 기존 도입부]**
안녕하세요, 염용주 변호사입니다.
오늘은 사립학교 교원징계위원회에 대해 알아보겠습니다.
사립학교에서 교원에 대한 징계를 진행할 때는 반드시 징계위원회를
구성해야 합니다. 이번 글에서는 징계위원회의 구성과 절차에 대해
상세히 살펴보겠습니다.

**[AFTER — 3단 도입부]**

📌 2026년 2월 기준 최신 법령 반영 (최초 작성: 20XX년 X월)

안녕하세요, 교육법 전문 변호사 염용주입니다.              ← ① 마이크로 인사
사립학교 교원징계위원회란 사립학교법 제61조에 따라        ← ① Direct Answer
교원 징계를 심의·의결하는 필수 기관으로, 위원 5인 이상
(외부위원 1/2 이상) 구성, 통보일로부터 30일 이내 의결이 원칙입니다.

갑작스러운 징계 통보에 당혹스러우실 겁니다.              ← ② 공감
소명 기회 전 준비가 결과를 좌우하므로,                  ← ② 손실회피
아래 내용을 꼼꼼히 확인해 보시길 권합니다.

이 글에서 아래 내용을 실무 경험을 바탕으로             ← ③ 글 안내 + 목차
정리해 드리겠습니다.

**목차**
1. 징계위원회 구성 요건
2. 징계 절차 단계별 정리
3. 징계 불복 방법과 기한

---

### 3단 도입부가 동시에 해결하는 것

| 목표 | 해결 방법 |
|------|---------|
| 스마트블록 미리보기 | ①의 첫 3줄에 정의+수치가 모두 포함됨 |
| AI 브리핑 추출 | "~란 ~입니다" 정의형 문장이 인사 직후 등장 |
| 독자 이탈 방지 | ②에서 감정을 인정하여 "이 글이 내 상황에 맞다" 확신 |
| 스크롤 유도 | ②의 손실 암시 → "안 읽으면 손해" 심리 작동 |
| 체류시간 증대 | ③에서 글 구조를 미리 보여주어 끝까지 읽을 동기 부여 |
| 자연스러운 톤 유지 | 인사말이 살아있어 블로그 특유의 인간미 보존 |
```

### [프롬프트 추가 섹션] STEP 2 — 본문 구조 재설계 (GEO + 체류시간)

```markdown
## STEP 2. 본문 구조 재설계

### 2-1. 소제목 체계 점검 — 의도 기반 재편

기존 소제목이 단순 나열형인 경우, 검색 의도(intent)별로 재구성합니다.

**변환 규칙:**

| 기존 패턴 | 개선 패턴 | 이유 |
|----------|----------|------|
| "1. 개요" | "교원징계위원회란? (구성·역할)" | 스마트블록 "정의" 블록 매칭 |
| "2. 절차" | "징계 절차, 어떤 순서로 진행되나요?" | 스마트블록 "절차" 블록 매칭 |
| "3. 주의사항" | "징계 통보를 받았을 때 반드시 확인할 3가지" | 스마트블록 "체크리스트" 블록 매칭 |
| "4. 판례" | "징계 취소된 실제 사례 — 00고등법원 2024나XXXXX" | 스마트블록 "사례" 블록 매칭 |

### 2-2. GEO 구조화 — 팩트 블록 삽입

각 소제목 섹션 내에 아래 형태의 **팩트 블록**을 1개 이상 삽입합니다:

---

**[팩트 블록 템플릿]**

> 📋 **핵심 정보 정리**
> - **근거 법령:** {법률명} 제{X}조 제{Y}항
> - **적용 대상:** {누구에게 해당하는지}
> - **기한:** {일수 또는 기간}
> - **위반 시:** {제재 내용}
> - **최종 확인일:** 2026년 {월}월 기준

---

**적용 이유:**
- 생성형 AI가 팩트를 추출할 때 명확한 키-값 쌍을 선호
- 사용자가 스크롤 중 시각적으로 멈추는 "앵커 포인트" 역할 (체류시간↑)
- 네이버 AI 브리핑이 정보를 요약할 때 이 형태를 우선 인용

### 2-3. Q&A 블록 삽입 (AEO 핵심)

기존 글의 주제에서 파생되는 **관련 질문 2~3개**를 Q&A 형태로 본문 중간~하단에 삽입합니다.

---

**[Q&A 블록 템플릿]**

> **Q. {잠재 의뢰인이 실제로 궁금해할 질문}?**
>
> A. {2~3문장의 명확한 답변}. {법률 근거}에 따르면 {구체적 내용}입니다.
> 다만, {예외 사항이나 주의점 1가지}도 확인이 필요합니다.
> 이 부분이 걱정되시면 [교육법 전문 변호사와 상담]을 권합니다.

---

**Q&A 질문 설계 원칙:**
- 네이버 "연관 검색어"와 "관련 질문" 참조하여 실제 검색어 기반으로 작성
- "~하면 어떻게 되나요?", "~해도 되나요?", "~비용은 얼마인가요?" 패턴 활용
- 변호사 광고 규정을 준수하면서도 상담 유도가 가능한 수준으로 작성
```

### [프롬프트 추가 섹션] STEP 3 — 리텐션(체류시간) 강화 장치

```markdown
## STEP 3. 리텐션 강화 장치

D.I.A+ 알고리즘은 체류시간과 스크롤 깊이를 핵심 지표로 활용합니다.
기존 글에 아래 장치를 추가하여 이탈률을 낮춥니다.

### 3-1. 훅(Hook) 강화 — 첫 화면 3초 법칙

기존 글의 첫 화면(스크롤 없이 보이는 영역)에 아래 중 1가지 이상을 삽입:

| 훅 유형 | 예시 | 삽입 위치 |
|---------|------|----------|
| **위기감 훅** | "이 조치를 놓치면 징계가 확정됩니다" | 제목 바로 아래 |
| **수치 훅** | "10건 중 7건은 초기 대응 실패가 원인" | Direct Answer 블록 내 |
| **공감 훅** | "갑자기 통보를 받으면 누구나 당황합니다" | 인사말 직후 |
| **실익 훅** | "이 글을 끝까지 읽으면 ~를 알 수 있습니다" | 목차 직전 |

### 3-2. 시각 요소 삽입 — 스크롤 유도

**2~3 스크롤마다 시각적 브레이크를 삽입합니다:**

- 📋 팩트 블록 (STEP 2에서 설계한 것)
- 📊 비교표 또는 절차 요약표
- 💡 실무 포인트 박스 ("염용주 변호사의 실무 포인트")
- ⚠️ 주의사항 박스
- 📞 중간 CTA ("이 부분이 해당되시면 상담 문의: 070-4149-1332")

**삽입 기준:** 본문 800자(약 2 모바일 스크롤)마다 시각 요소 1개

### 3-3. 내부 링크 전략 — 이탈을 다른 글로 전환

기존 글에 **관련 글 링크를 3~5개** 삽입합니다.

**내부 링크 삽입 패턴:**

```
📌 관련 글 안내
- 이 사안의 전체 절차가 궁금하시면 → [학교폭력 사안처리 절차 총정리]
- 가해학생 측 대응 방법 → [학교폭력 가해자 부모가 알아야 할 5가지]
- 불복(재심) 절차 → [학교폭력 재심 청구, 이렇게 진행됩니다]
```

**링크 위치 전략:**
- 본문 중간 (1/3 지점): 관련 배경 지식 글 1개
- 본문 2/3 지점: 심화 주제 글 1개
- 글 하단 (마무리 전): 시리즈 다음 편 또는 상담 안내 글 2~3개

### 3-4. 목차(인덱스) 추가

기존 글에 목차가 없는 경우, 본문 상단(Direct Answer 블록 아래)에 클릭 가능한 목차를 추가합니다.

```
📑 이 글의 내용
1. 교원징계위원회란?
2. 징계 절차 단계별 정리
3. 징계 사유별 양정 기준
4. 불복 절차와 주의사항
5. 실제 취소 사례
6. 자주 묻는 질문(Q&A)
```

→ 목차가 있으면 사용자가 "원하는 정보가 있다"는 확신을 갖고 스크롤합니다.
```

```markdown
## STEP 4. 법령·판례 최신화

### 4-1. 최신화 체크리스트

기존 글을 열 때 아래 항목을 반드시 확인합니다:
```

```
### 4-3. 법령 변경 이력 박스
```

- 기존: 학교폭력예방법 제17조 제1항 — {이전 내용 요약}
- 현행: 2025년 X월 개정 — {변경 내용 요약}
- 영향: {실무적으로 달라진 점 1~2문장}
```

→ 이 박스는 GEO 최적화에도 기여합니다 (AI가 "변경 이력"을 구조적으로 파악)
```

```
   └── 법령·판례 변경사항 확인 (대법원 종합법률정보)
   │
[매월 2~3주차] 업데이트 실행 (4~6편/월)
   │
   ├── B등급: 2~3편 풀 업데이트 (STEP 1~6 전체)
   ├── A등급: 1~2편 리텐션+AEO 보강 (STEP 1, 3 중심)
   └── C등급: 1~2편 제목·도입부 개선 (STEP 1 중심)
   │
[매월 4주차] 성과 확인
   │
   ├── 업데이트 전후 조회수·체류시간 비교
   ├── 스마트블록 노출 여부 확인
   └── 다음 달 업데이트 대상 사전 선정
```

### 프롬프트 입력 예시

```
[역할] 교육법 전문 변호사 염용주의 블로그 글 업데이트 담당 에디터

[모드] 업데이트 모드 활성화

[대상 글]
- 제목: {기존 글 제목}
- 현재 본문: {기존 본문 전체 붙여넣기}

[업데이트 지시]
1. STEP 1(도입부 리엔지니어링) 적용 — Direct Answer 블록 삽입
2. STEP 2(본문 구조 재설계) 적용 — 소제목 의도 기반 재편 + 팩트 블록 + Q&A
3. STEP 3(리텐션 강화) 적용 — 훅 + 시각 요소 + 내부 링크 + 목차
4. STEP 4(법령 최신화) 적용 — 2026년 2월 기준 확인
5. STEP 5(CTA) 적용 — 3단 CTA 배치
6. STEP 6(자가진단) 실행 — 체크리스트 결과 하단에 첨부

[제약 조건]
- 기존 URL 유지 (새 글 발행 아님)
- 핵심 키워드 변경 금지: {기존 키워드 나열}
- 변호사 광고에 관한 규정 (2025년 개정) 준수
- 기존 내용 삭제 최소화, 추가·재구성 우선

[출력 형식]
- 업데이트된 전체 본문
- 변경 사항 요약 (추가/수정/삭제 목록)
- 자가진단 체크리스트 결과
```
