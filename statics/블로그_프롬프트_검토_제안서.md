## 1. 🚨 즉시 수정이 필요한 항목

### 1-1. 변호사 광고 규정 업데이트 (긴급)

현재 프롬프트는 "변호사업무광고에관한규정 제3조~제5조"를 참조하고 있지만, 대한변호사협회는 **2025년 2월 6일** 및 **2025년 6월 30일**에 규정을 대폭 개정하여 명칭 자체가 **「변호사 광고에 관한 규정」**으로 변경되었습니다.

**주요 변경사항:**

| 항목 | 기존 | 개정 후 (2025) |
|------|------|---------------|
| 규정명 | 변호사업무광고에관한규정 | 변호사 광고에 관한 규정 |
| '전문'/'전담' 표현 | 사용 제한적 | 주로 취급하는 업무에 한해 사용 가능 (제7조) |
| 무료상담/할인 광고 | 제한적 금지 | **명확히 금지** |
| AI 관련 | 규정 없음 | **AI 사용 광고 시 협회 등록 필수, 검토 변호사 이름 기재 의무** |
| 환불 약속 | 명시적 규정 없음 | **"기각 시 100% 환불" 등 금지** |
| 수임료 할인 홍보 | 제한적 | **명확히 금지** |

**→ 조치:** 프롬프트의 "변호사업무광고규정" 섹션 전체를 2025년 개정 규정에 맞춰 재작성 필요. 특히 AI 관련 신설 조항은 블로그 콘텐츠 제작 시에도 참고해야 합니다.

### 1-2. 날짜/연도 업데이트

- "최종 업데이트: 2024년 12월" → **2026년 2월**로 변경
- 제목 예시 "2024년 학폭법 달라진 점" → **2026년** 기준으로 변경
- 전반적으로 연도 레퍼런스를 현행화할 것

## 2. 🔄 알고리즘/SEO 섹션 대폭 보강 필요

### 2-1. 스마트블록(Smart Block) 대응 전략 추가

네이버는 2024~2025년 사이에 **VIEW 탭을 폐지**하고 **스마트블록** 시스템으로 완전히 전환했습니다. 이제 검색 결과가 주제 의도별로 분해되어 여러 블록으로 나뉘어 표시됩니다.

**프롬프트에 추가해야 할 내용:**

```
### 네이버 스마트블록 최적화

- 하나의 글에서 **하나의 명확한 검색 의도**에 답변하도록 작성
- 글 제목과 소제목이 스마트블록의 분류 키워드와 일치하도록 설계
- 예: "학폭 가해자 전학 조치"를 다룰 때, "전학 조치 요건",
  "전학 조치 불복 절차", "전학 조치 사례" 등 세부 의도별로
  소제목을 구성
- 첫 문단 2~3줄 안에 핵심 답변을 제시 (스마트블록 미리보기에 노출)
```

### 2-2. AEO 섹션 대폭 강화

현재 프롬프트의 AEO 섹션이 너무 간략합니다. 2025~2026년 현재 **AI 브리핑**, **제로클릭 검색** 비중이 급격히 증가(60% 이상)하고 있어 AEO는 독립적인 주요 섹션으로 승격시켜야 합니다.

**추가 권장 내용:**

```
### AEO (Answer Engine Optimization) 전략 - 확장

#### AI 브리핑 최적화
- 네이버 AI 브리핑이 답변을 추출할 수 있도록 **정의형 문장** 작성
  예: "학교폭력예방법 제17조에 따른 전학 조치란, ~를 의미합니다."
- 첫 문단에 질문에 대한 **직접적 답변(Direct Answer)** 배치
- 법령 조항 번호와 함께 구체적 수치/기한 명시

#### GEO (Generative Engine Optimization)
- 생성형 AI가 콘텐츠를 이해하고 재구성할 수 있는 구조 설계
- 팩트 기반의 명확한 정보 제공 (모호한 표현 최소화)
- 출처를 명시하여 AI가 신뢰도를 판단할 수 있도록 함

#### 구조화된 답변 포맷
- "~란 무엇인가?" → 정의 1~2문장 → 상세 설명 순서
- 절차 설명 시 반드시 번호 매기기 (1단계, 2단계...)
- 기한/비용/요건 등 핵심 정보는 표로 정리
```

### 2-3. AI 생성 콘텐츠 감지 대응

네이버가 AI 자동생성 콘텐츠 필터링을 강화하고 있으므로, 프롬프트에 다음 가이드라인을 추가해야 합니다:

```
### AI 콘텐츠 감지 회피 (자연스러운 글쓰기)

⚠️ 네이버는 AI 자동생성 콘텐츠에 대한 필터링을 강화하고 있음

- 변호사 본인의 **실제 경험과 의견**을 반영한 문장 포함
  예: "제가 실무에서 자주 마주하는 사례인데요..."
- 기계적으로 반복되는 문장 구조를 피하고 문체에 변화를 줄 것
- 지나치게 완벽한 구조보다 자연스러운 흐름을 우선시
- 실제 상담 사례를 바탕으로 한 구체적 디테일 포함
```

## 3. 📱 모바일 최적화 섹션 신설 필요

현재 프롬프트에 **모바일 최적화 가이드가 전혀 없습니다.** 네이버 블로그 트래픽의 70% 이상이 모바일에서 발생하므로 이 섹션은 필수입니다.

```
### 모바일 최적화 가이드

#### 가독성
- 한 문단 최대 3~4줄 (모바일 기준), 5문장 미만
- 중요 정보는 **굵은 글씨**로 강조하되 남용 금지
- 문단 사이 적절한 여백 (이미지 또는 구분선 활용)

#### 이미지
- 썸네일 권장 사이즈: 1300 x 885px
- 본문 이미지: 가로 폭 최소 960px 이상
- 이미지 개수: 6~13개 권장 (현재 프롬프트는 5개로 한정)
- 최신 촬영/제작 이미지 사용 (네이버가 이미지 생성 시점 감지)

#### 스크롤 유도
- 3스크롤(모바일 기준 약 800자) 이내에 핵심 정보 배치
- 소제목을 통해 원하는 정보로 빠르게 이동할 수 있는 구조
```

## 4. 📊 분량 기준 재검토

### 4-1. 글자수 기준

현재 프롬프트의 **2,500~3,000자(공백 제외)** 기준은 일반적인 범위이지만, 법률 전문 블로그의 경우 경쟁 키워드 상위노출을 위해 **3,000~4,000자**가 더 유리하다는 분석이 있습니다.

**제안:** 기본 기준은 유지하되, 다음과 같이 키워드 경쟁도에 따른 유동적 기준을 추가

```
| 키워드 경쟁도 | 권장 글자수 (공백 제외) | 이미지 수 |
|-------------|---------------------|----------|
| 낮음 (월 검색량 500 미만) | 2,000~2,500자 | 5~8개 |
| 중간 (월 검색량 500~3,000) | 2,500~3,000자 | 8~12개 |
| 높음 (월 검색량 3,000 이상) | 3,000~4,000자 | 10~15개 |
```

### 4-2. 이미지 개수

현재 프롬프트는 5개 이미지만 지정하고 있지만, 네이버 상위노출 기준으로는 **6~13개**가 권장됩니다. 법률 블로그 특성상 본문 중간중간에 정보를 정리한 **인포그래픽, 표 캡처, 절차 다이어그램** 등을 추가 배치하는 것이 좋습니다.

**제안:** 기존 5개 + 추가 이미지 유형 정의

```
| 추가 이미지 유형 | 위치 | 예시 |
|---------------|------|------|
| 절차 플로우차트 | 본론2 | 학폭위 진행 절차 다이어그램 |
| 핵심 정보 요약 카드 | 각 본론 말미 | "꼭 기억하세요" 요약 이미지 |
| 관련 법령 하이라이트 | 본론1 | 법 조항 핵심 부분 강조 이미지 |
| 체크리스트 이미지 | 결론 | 대응 체크리스트 인포그래픽 |
```

## 5. 🔗 내부 링크 전략 추가

현재 프롬프트에 **내부 링크(다른 블로그 글로의 링크) 전략이 없습니다.** 내부 링크는 체류시간 증가와 C-RANK 점수 향상에 직접적으로 기여합니다.

```
### 내부 링크 전략

- 본문 내 관련 주제의 이전 포스팅 **2~3개** 자연스럽게 링크
- 링크 위치: 본론2 또는 본론3에서 "관련 내용은 아래 글을 참고해 주세요"
- 시리즈 글의 경우 "이전 글 보기 / 다음 글 예고" 구조 활용
- 링크 텍스트에 해당 글의 핵심 키워드 포함
  예: "학폭위 절차에 대한 자세한 내용은 [학폭위 절차 완벽 가이드] 글을 참고하세요"
```

## 6. ✍️ 톤/어조 가이드 보강

### 6-1. 공감 표현 다양화

현재 프롬프트의 서론 공감 문장 예시가 한정적입니다. 다양한 상황별 공감 표현 레퍼토리를 추가하면 좋겠습니다.

```
### 상황별 공감 표현 예시

| 상황 | 공감 표현 |
|------|----------|
| 학폭 피해 학부모 | "아이가 다쳤다는 소식에 밤잠을 이루지 못하셨을 겁니다" |
| 학폭 가해 학부모 | "당장 아이의 미래가 걱정되시는 마음, 잘 알고 있습니다" |
| 징계 대상 교원 | "오랜 교직 생활에서 이런 상황을 마주하셨을 때의 당혹감이 느껴집니다" |
| 학생부 정정 요청 | "기록 하나가 아이의 진학에 영향을 줄 수 있다는 불안감, 충분히 공감합니다" |
```

### 6-2. 전문성-친근함 밸런스 기준

현재 프롬프트가 "따뜻하고 친근한 어조"를 강조하지만, 법률 전문 블로그에서 **신뢰감을 해치지 않는 선**에 대한 구체적인 기준이 부족합니다.

```
### 어조 밸런스 가이드

✅ 적절한 표현:
- "이 부분은 법적으로 중요한 의미를 갖습니다" (전문적 + 이해 가능)
- "쉽게 말씀드리면, ~라고 보시면 됩니다" (전환 표현)

❌ 피해야 할 표현:
- "법적으로 말하자면 blah blah..." (지나치게 격식체)
- "걱정 마세요~ 다 잘 될 거예요!" (지나치게 가벼움)
- "헐, 이거 진짜 중요한데요" (지나치게 캐주얼)
```

## 7. 🔍 글 작성 전 조사 섹션 보강

### 7-1. 키워드 리서치 절차 추가

현재 프롬프트에는 법률/판례 조사는 있지만 **키워드 리서치 절차**가 빠져 있습니다.

```
### 키워드 리서치 (글 작성 전 필수)

1. 네이버 키워드 도구(또는 블랙키위 등)로 검색량 확인
2. 메인 키워드 + 세부 키워드(롱테일) 2~3개 선정
3. 해당 키워드의 상위 5개 포스팅 분석:
   - 글자수, 이미지 수, 구조 파악
   - 빠져 있는 정보(정보 갭) 식별
4. 연관 검색어 / 자동완성 키워드 수집
```

### 7-2. 경쟁 콘텐츠 분석

상위노출을 목표로 한다면 경쟁 포스팅 분석은 필수인데, 현재 프롬프트에 이 과정이 없습니다.

```
### 경쟁 콘텐츠 분석 (선택)

- 검색 결과 상위 3~5개 글의 구조, 분량, 이미지 수 파악
- 해당 글들이 다루지 않는 **정보 갭(Information Gap)** 발견
- 차별화 포인트 1~2가지 설정
  예: "다른 글에서 다루지 않는 최신 판례 추가",
      "실제 절차 경험 기반 타임라인 제공"
```

## 8. 📋 기타 개선 사항

### 8-1. 시리즈/연재물 전략 부재

C-RANK에서 "주제 전문성"이 40%를 차지하므로, 단일 글이 아닌 **시리즈 연재 전략**에 대한 가이드라인이 필요합니다.

```
### 시리즈 연재 전략

- 하나의 큰 주제를 3~5편으로 나누어 연재
  예: "학교폭력 완벽 가이드" → ①개념편 ②절차편 ③대응편 ④판례편
- 각 글 서론에 시리즈 안내, 결론에 다음 편 예고
- 시리즈 간 상호 내부 링크 필수
```

### 8-2. 발행 시간/빈도 가이드 부재

```
### 발행 전략

- 권장 발행 시간: 오전 6~8시 또는 오후 12~14시 (직장인 통근/점심 시간)
- 발행 빈도: 주 2~3회 (꾸준한 활동성이 C-RANK에 긍정적)
- 주제 집중도: 교육법/학교폭력 관련 글이 전체의 70% 이상 유지
```

### 8-3. 댓글/공감 유도 전략 부재

사용자 반응(댓글, 공감)이 D.I.A 점수에 영향을 미치므로:

```
### 독자 참여 유도

- 결론부에 **질문형 마무리** 활용
  예: "비슷한 상황에서 고민이 있으시다면 댓글로 남겨 주세요"
- 다만, 과도한 댓글 유도는 지양
  ❌ "댓글 달아주시면 ~해 드립니다" (변호사 광고 규정 위반 소지)
```

### 8-4. 이미지 프롬프트 섹션 보완

현재 영문 프롬프트만 제시하고 있으나, **이미지 스타일/규격** 기준이 없습니다.

```
### 이미지 생성 가이드 보완

- 스타일: 따뜻한 일러스트 또는 클린한 3D 렌더링 (사진 스타일 지양)
- 해상도: 최소 1300 x 885px (네이버 썸네일 최적 사이즈)
- 색감: 신뢰감을 주는 네이비/블루 계열 + 따뜻한 포인트 컬러
- 텍스트 포함 여부: 본문 이미지에는 핵심 키워드 텍스트 오버레이 권장
- ⚠️ 실제 법원/학교 사진 사용 시 초상권·저작권 주의
```

### 8-5. 품질 체크리스트 항목 추가
