# 네이버 블로그 컨텐츠 업데이트 프로젝트 PRD

## 프로젝트 개요
- 프로젝트명: 교육법 전문 블로그 성장 전략 실행
- 블로그: 염용주 대표변호사의 교육법 쌀롱
- 목표: 교육법 분야 1위 법률 블로그로 성장

---

## Phase 1: 즉시 실행 (1-2주 내)

### Task 1: 발행 일정 캘린더 수립
- 주 2-3회 정기 발행 스케줄 작성
- 화요일, 목요일 오전 9-10시 발행 시간 설정
- 월간 컨텐츠 플랜 템플릿 제작
  - 1주: 법률 가이드 + 판례 분석
  - 2주: 시사 해설 + Q&A
  - 3주: 법률 가이드 + 사례 분석
  - 4주: 정책 해설 + Q&A

### Task 2: 신규 시리즈 첫 글 작성 - "교육법 판례 읽기"
- 최신 대법원/행정법원 판례 선정
- 판례 분석 글 작성
- 타겟 독자: 교원, 교육행정가
- SEO 최적화 태그 적용

### Task 3: 기존 인기 글 업데이트
- 인기 게시글 목록 파악
- 2024년 법령 변경 사항 반영
- 최신 정보로 컨텐츠 갱신

---

## Phase 2: 단기 (1개월 내)

### Task 4: 인포그래픽/체크리스트 템플릿 제작
- 학폭위 준비 체크리스트 템플릿
- 교원 징계 대응 체크리스트 템플릿
- 카드뉴스 형태 이미지 템플릿
- 인포그래픽 요약본 템플릿

### Task 5: 이웃 블로거 네트워크 확장
- 교육 관련 블로거 10명 이상 탐색
- 이웃 추가 및 교류 시작
- 댓글 교류 활성화

### Task 6: 태그 전략 재정비
- 롱테일 키워드 확장
  - 서울교육청변호사, 경기교육청징계
  - 교사고소당했을때, 학폭가해자대응
  - 교원소청심사기간, 학교폭력조치불복
- 지역 타겟팅 키워드 추가
- 태그 10-15개 조합 전략 수립

### Task 7: 신규 시리즈 - "학부모가 알아야 할 교육법" 시작
- 학교폭력, 학교안전 관련 컨텐츠 기획
- 학부모 관점 글 2회 작성
- 월 2회 발행 일정 수립

### Task 8: 신규 시리즈 - "교원 Q&A" 시작
- 실제 상담 사례 익명화 기획
- 현직 교원 타겟 Q&A 글 2회 작성
- 월 2회 발행 일정 수립

---

## Phase 3: 중기 (3개월 내)

### Task 9: 모든 시리즈 3회 이상 발행 완료
- "교육법 판례 읽기" 시리즈 3회 이상
- "학부모가 알아야 할 교육법" 시리즈 3회 이상
- "교육청 공무원을 위한 법률 가이드" 시리즈 시작 및 3회 발행
- "교원 Q&A" 시리즈 3회 이상

### Task 10: 블로그 방문자 수 50% 증가 달성
- 현재 방문자 수 기준선 측정
- SEO 최적화 효과 모니터링
- 주간/월간 방문자 추이 분석
- 개선 포인트 도출 및 적용

### Task 11: 상담 예약 전환율 측정 시스템 구축
- CTA 클릭 추적 설정
- 상담 예약 전환 경로 분석
- 전환율 5% 달성 목표 설정
- A/B 테스트 진행

### Task 12: 인게이지먼트 향상 활동
- 글 말미 질문 유도 문구 표준화
- 댓글 성실 답변 체계 구축
- 공감/댓글 수 목표 달성 (공감 10개, 댓글 5개)

---

## Phase 4: 시즌별 컨텐츠 대응

### Task 13: 2025년 12월 시즌 컨텐츠
- 사교육카르텔 후속 대응 글 작성
- 학교생활기록부 정정 청구 가이드 작성

### Task 14: 2026년 1분기 시즌 컨텐츠
- 교원 휴직/복직 관련 법률 가이드
- 2026년 교육법 개정 전망 기획 글
- 2-3월 신학기 학폭 대응 가이드

---

## 성과 측정 지표 (KPI)

| 지표 | 현재 | 3개월 목표 | 6개월 목표 |
|------|------|-----------|-----------|
| 월간 방문자 | 기준선 | +50% | +100% |
| 게시글당 공감 | 3-5개 | 10개 | 20개 |
| 댓글 수 | 0-2개 | 5개 | 10개 |
| 상담 예약 클릭 | 측정 필요 | 측정 시작 | 전환율 5% |
| 네이버 검색 노출 | - | 상위 10 | 상위 5 |
