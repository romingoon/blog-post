## 발행 시 주의사항

### 1. 발행 시간 준수
- **화요일/목요일/토요일 오전 9시** 정기 발행 (주 3회)
- 주요 연재 시리즈는 연속 발행으로 독자 관심 유지

### 2. 컨텐츠 구성 요소
- **도입부**: 독자의 고민/문제 상황 제시
- **본문**: 법률 해설 + 실무 조언 (5-7개 섹션)
- **사례**: 실제 승소 사례 또는 상담 사례 (익명화)
- **체크리스트**: 핵심 내용 요약 (3-5개 항목)
- **CTA**: 법률상담 예약 링크, 댓글 유도 질문

### 3. SEO 최적화 체크리스트
- [ ] 제목 50자 내외, 핵심 키워드 포함
- [ ] 메타 설명 100-150자 작성
- [ ] 태그 10-15개 (핵심+롱테일 조합)
- [ ] 이미지 alt 텍스트 추가
- [ ] 내부 링크 2-3개 (관련 이전 포스트)

### 4. 독자 인게이지먼트 전략
- 글 말미에 질문 유도: "이런 경험이 있으신가요? 댓글로 공유해주세요"
- 시리즈 다음 편 예고: "다음 주에는 ○○에 대해 다룹니다"
- 체크리스트/인포그래픽 이미지 제작 (공유 유도)

### 5. 법률 윤리 준수
- 특정 사건 명시 금지 (익명화 철저)
- 법률 자문으로 오해받지 않도록 disclaimer 추가
- 정확한 법령 인용 및 최신 개정 내용 반영
